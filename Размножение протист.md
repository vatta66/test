- Агамогония (бесполое размн.)
	- А - отрицание, гамос - брачный процесс (безбрачный)
	- Клетка, претерпевающая бесполое размножение - агамонт
		- То есть любая клетка, которая сейчас делится бесполым путем
	- Виды
		- [[Митоз]]
			- ? У протистов его 6 типов
		- Палинтомия
			- Серия последовательных делений без роста потомков
			- Как для моно-, так и для полиэнергидов
		- Шизогония
			- Только для моноэнергидов
			- Процесс
				- Сначала делится ядро - кариотомия
					- Томос - резать (патологоанатом - тот кто режет трупов)
				- Затем плазмотомия - множественное деление цитоплазмы
					- Получается много мелких одноядерных клеток
			- Шизо - разделение
				- Как и [[Шизофрения]]
		- Плазматомия
			- Для полиэнергидных протист
			- Цитоплазма делится, ядра распределяются между клетками (не поровну)
		- ![[Размножение протист-10-09-2024.png|300]]
- Гамогония (половое)
	- Клетка, размножающаяся половым путем - гамонт
	- Изогамия - слияние одинаковых гамет
	- Анизогамия - слияние отличающихся друг от друга по размеру гамет
	- Оогамия - несколько маленьких гамет со жгутиком копулируют с одной большой гаметой без жгутика
		- Маленькие гаметы - микрогаметы
		- Крупная - макрогамета
