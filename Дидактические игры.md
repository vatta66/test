# Дидактические игры

- Специально созданные ситуации, моделирующие реальность, из которых учащимся предлагается найти выход
- Виды игр
	- Симуляционные
		- Способствуют воспроизведению опред. качества
		- Учащиеся приспосабливаются к опред. анализу проблем
			- Например, симуляция заседания парламента, в ходе которой ученики изучают его работу
	- Инсценировки
		-  Например заранее подготовленный диалога, дискуссии на определенную тему, форму театрализованного воспроизведения событий и тд
- Мои мысли по поводу дидактических игр - [[Геймификация уроков]]
