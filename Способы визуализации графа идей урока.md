# Способы визуализации графа идей урока

-  In each way we create illustrative [[Кластер знаний]] to help students to see relationships between ideas throughout the entire lesson
	- It make students more interested in lesson([[Познавательный интерес]]) according to the [[Искусственная актуализация знания]]
- For [[Эвристическая беседа]]
	- In the beginning, we have blank sheet. Then, we will add ideas and links between them as they are revealed through heuristic learning
		- Students also can reveal connections between the ideas by themselves
- For standart lesson
	- We can create “uncreated ideas” and link new existed one with them as it described in [[Представление кластеров знаний уроков как вершин графа как средство систематизации знаний ученика]]
	- We can make links only with existed ideas on board and add more links as new ideas emerge
		- Можно улучшить этот метод, показывая цветом новые знания в кластере и связи со старыми
		- Пример - [[Системы вуда по водорослям canvas.canvas|Системы вуда по водорослям canvas]]
	- Отдельную подтему представлять в видео отдельных системок на канвасе, но связывать их с уже созданными
		- Пример - [[Системы урока Корень растения canvas.canvas|Системы урока Корень растения canvas]]
