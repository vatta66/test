# Посттрансляционная модификация белка
Up: [[Белки]], [[Трансляция]]
Next:

- Белок принимает свою третичную структуру с помощью шаперонов
	- Если полипептид свернулся неправильно, на него вешают цепочку из убиквитинов (полиубиквитин) - маркер для протеасомы для разрушения этого полипептида
- Соединяется с кофактором
- Подвергается ковалентным модификациям
	- Примеры
		- В 70% случаев метионин с начала полипептида вырезается, а этот конец ацетилируется
		- [[Фосфорилирование белков]]
		- [[Убиквитин|Убиквитинирование]] и Сумоилирование
		- [[Метилирование]] или ацетилирование лизина в [[гистоны|гистонах]]
		- Пальмитирование цистеина
			- характерно для [[Клеточная мембрана|мембранных]] белков
		- Добавление ацетилглюкозамина
	- Эти модификации можно комбинировать, создавая такой код
- Соединяется с другими полипептидами в рабочий белок
