# Зоны [[Корень растения]]

1. **Корневой чехлик** - на конце корня - обеспечивает защиту зоны деления (т.к. ее клетки постоянно делятся, они очень нежные)
2. **Зона деления** - состоит из маленьких клеток, которые постоянно делятся, обеспечивающих [[рост корня]], сост. из [[Образовательная ткань растения]]
3. **Зона роста** - маленькие клетки зоны деления быстро растут
4. **Зона всасывания** - корневые волоски, всасывает минер. в-ва - [[Минеральное питание корня]]. Эти вещества будут поступать по древесным каналам в [[стебель]], откуда по [[Жилки листа]] поступают в [[лист]]. Это - зона дифференциации
5. **Зона проведения** - транспорт в-в. От нее отходят боковые корни. Реализация функции заякоривания - [[Функции корня]]
