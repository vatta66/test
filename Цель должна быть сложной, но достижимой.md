# Цель должна быть сложной, но достижимой

- [[Теория постановки целей Локка]] утверждает, что
- Цель должна быть сложной
- Но, если цель слишком сложная - она вообще не будет достигнута
- Если слишком проста - будет [[скука|скучно]]
	- Это можно сравнить с условиями для входа в [[Состояние потока]]
- Таким образом, повышение сложности достижения цели будет увеличивать [[Мотивация]] только если они остаются реальными для выполнения
	- Тажке можно сказать о том, что человек должен ощущать, что способен достичь этой цели, если приложит все усилия
		- Об этом же говорит [[Мотивационная теория ожиданий]])
- Для реализации этого условия [[Цель должна быть конкретной]]
