# Варианты тагмозиса ракообразных
Up: [[Ракообразные]]
Next:

- Самый древний
	- Цефалон
	- Торакс (1-40 сегментов)
		- Конечности на тораксе - переоподы
	- Абдомен (1-10 сегментов)
		- Бывают конечности - плеоподы
	- Тельсон (анальная лопасть)
		- Бывают парные придатки - фурки
	- ![[Ракообразные-20-03-2025.png|300]]
- Второй
	- Цефалон может сливаться с частью сегментов торекса - цефалоторакс (головогрудь)
		- 1-3 сегмента от груди
			- Конечности этиэ сегментов - ногочелюсти (максиллоподы)
	- Торакс
	- Абдомен
	- Тельсон с фурками
- Третий
	- Цефалон разделяется на два отдела
		- Протоцефалон - акрон + 3 сегмента
		- Гнатоцефалон - 3 сегмента
			- Может сростаться с несколькими сегментами торакса - гнатоторакс (челюстегрудь)
	- Дальше без изменений
