# Половой отбор
Up:
Next:

- Проводится одним полом в отношении другим
	- Тот пол, который тратит больше ресурсов на выращивание потомства, и будет вести отбор
	- Зачастую это женский пол (см. [[Конкуренция полов при размножении]])
- Определить, какой самец имеет лучшие [[ген]]ы, позволяет [[Принцип гандикапа]]
- Половой отбор контролируется [[Естественный отбор|естественным отбором]]
	- Т.к. если самец концентрируется только на развитии признаков, увеличивающих его привлекательность для самки, он теряет [[Приспособленность]] к условиям среды
