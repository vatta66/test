# Закон Йоста

- Регулярное повторение на протяжении долгого времени (распределенное) эффективней большого количества повторений за маленький промежуток времени (концентрированное)
- Из двух ассоциаций равной силы, где одна из них более старая, при повторениях будет усиливаться старая ассоциация - [[Ассоциативное обучение]]
