# Коммерческие банки
Up: [[Финансовый рынок]]
Next: [[Центральный Банк РФ]]

- **Коммерческие банки** - негосударственные кредитные учреждения, осуществляющие банковские операции по обслуживанию физических и юридических лиц
- Функции коммерческих банков
	- Привлечение [[депозит]]ов
	- Выдача [[кредит]]ов
	- Расчетно-кассовые операции
	- Управление [[Вещь в праве|имуществом]] клиентов
	- Межбанковские операции: с иностранной валютой, с ценными бумагами, перевод денег
- Банковские операции
	- Кредитование
	- Потребительские кредиты
	- Валютные операции с [[Ценные бумаги]]
	- Валютные операции с драгоценными металлами
	- Ипотека
- Виды коммерческих банков
	- По форме собственности
		- [[Государство|Государственные]]
		- Акционерные
		- Кооперативные
		- Совместные с участием иностранного капитала
	- По характеру операций
		- Универсальные (все банковские операции)
		- Специализированные
	- По хозяйственному признаку
		- Промышленные
		- Торговые
		- Сельскохозяйственные
	- По сфере обслуживания
		- Региональные
		- Межрегиональные
		- Национальные
		- Международных
