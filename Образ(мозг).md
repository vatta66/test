# Образ

- На каждый образ существует несколько [[Распознающий модуль(мозг)]]
	- Некоторые повторяют друг друга 
	- Некоторые распознают один образ, но в разных положениях
- Состоит из 
	- Входной сигнал из [[Распознающий модуль(мозг)|модулей]]  низшего порядка
		- [[Синапс|Аксон]] от модуля, распознающего букву “А”, связан с множеством  [[Синапс|дендритов]] модулей, распознающих слово, где есть буква “А”
	- Имя образа
		- Аксон, выходящий из каждого модуля. 
			- Возбуждение аксона - распознавание соответствующего образа
	- Набор образов более высокого порядка, в состав которых он входит
		- Для буквы “а” - все слова с буквой “а”
- Характеризуется множеством входных сигналов, процессом распознования образа в модуле и выходным сигналом, поступающим к модулю более высокого порядка
