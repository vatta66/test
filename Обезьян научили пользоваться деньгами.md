# Обезьян научили пользоваться деньгами
Up: [[Деньги]] - [[Обезьяны]]
Next: [[Изучение психики шимпанзе В.Келера]]

- Когда обезьяны освоили концепцию денег, они начали
	- Откладывать их (копить)
	- Демонстрировать [[Импульсивность]] в их трате (делать импульсивные покупки)
	- Пытаться украсть монеты других
	- Одна обезьяна даже предложила монету другой за [[секс]]
	- Демонстрировать [[неприятие потерь]]
		- При возможности получить либо небольшую гарантию, либо шанс на более крупную награду, обезьяны, как правило, выбирали менее рискованный вариант - [[Переоценка малых вероятностей]]
