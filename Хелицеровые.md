# Хелицеровые
Подтип Helicerata из типа [[Членистоногие]]

- Две тагмы
	- Просома (головогрудь)
		- Акрон + 6 сегментов
		- Придатки
			- Глаза (простые)
			- хелицеры
				- Измельчение добычи
				- Гомологичны антеннам 1 [[Ракообразные]]
			- Педипальпы
				- Захват добычи, движение, сенсорная ф-ция, даже для переноса сперматофора
				- Не гомологичны антеннам 2, происхождение непонятно
			- 4 пары ходильных конечностей
	- Бывает промежуточный сегмент - интеркалярный
		- Чаще всего есть только на эмбриональных стадиях
	- Опистосома (брюшко)
		- 10-12 сегментов, иногда утрачена полностью
		- Подотделы
			- Мезосома (6 сегментов)
				- На 1 сегменте - половые крышки (оперкулюм) - прикрывают половое отверстие
				- Паутинные бородавки, легочные мешки, трахеи и др.
			- Метасома (4-6 сегментов)
				- Без придатков
	- Тельсон
		- Не у всех
- Дыхание
	- Жабры
		- Складки кожи с большим количеством кровеносных сосудов
		- Тонкая кутикула для облегчения газообмена
		- Эффективна в воде благодаря усиленному газообмену
			- На суше это становится минусом, т.к. через него животное начинает быстро терять воду
	- Легочные мешки
		- Те же жабры, только с почти закрытым входом в жаберную полость
		- Таким образом, водяные пары из тела уходят очень медленно, а воздух в эту полость проникает хорошо
- Одни из первых в палеозойскую эру вышли из воды и освоили сушу
- Классы
	- Класс [[Паукообразные]] (Arachnida)
	- Класс [[Мечехвосты]] (Xiphosura)
	- Класс Морские пауки (Pantopoda) 
	- Класс Ракоскорпионы (Eurypterida) (вымерший)
		- Появились в [[Ордовикский период|ордовикском периоде]] и вымерли в пермском
			- Пик - [[силурийский период]]
				- Могли достигать 3м
					- Но обычный размер - около 10 см
