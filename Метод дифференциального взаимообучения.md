# Метод дифференциального взаимообучения

- Учитель объясняет новый материал
- Ученикам даются задания сложности А, Б, С
- По выполнении задания ученики распределяются по трем группам в зависимости от кол-ва решенных задач
- Ученики из группы А объясняют ученикам из группы Б их ошибки
	- Ученики из группы Б объясняют ученикам из группа С их ошибки
- На этом первый цикл заканчивается
