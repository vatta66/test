# Используй триггеры для создания привычек

- [[Привычка]] легче всего внедрить, когда она следует сразу после определенного триггера
	- Т.к. привычка должна выполняться без направления [[внимание]] на факт необходимости ее выполнения - [[система 1]]
	- Нужно сделать так, чтобы действие начиналось автоматически, а это легче всего осуществить в форме автоматической реакции на определенное внешнее раздражение(триггер)
- Триггер эффективнее делать заметный, т.к. [[Отличительная черта притягивает внимание]]
- Например
	- Вы можете установить привычку медитировать после уже установленной привычки(чистить зубы, бегать по утрам и тд)
	- Вы можете установить привычку работать при наличии определенного триггера - [[Привычка изменять психологическое состояние после триггера]]
- Практические советы
	- Всегда слушайте определенную [[музыка]] только при работе
	- Включайте настольную лампу только при работе
	- Жуйте определенный вкус жевачки только при работе - [[Жевание жевачки повышает концентрацию]]
- Эти советы можно применить для того, чтобы добиться [[Состояние идейной осознанности]]
