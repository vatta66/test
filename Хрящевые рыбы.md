# Хрящевые рыбы
Up: [[Рыбы]]
Next: [[Костные рыбы]]

- Группы
	- Акулы
	- Скаты
	- [[Химеры]]
- Скелет
	- Ранние бесчелюстные (остракодермы) имели скелет, поэтому хрящи хрящевых - это производное костей
	- Есть хорда
- Внешнее строение
	- Брызгальце (спиракул)
		- Отверстие над глазами, в которое засасывается вода и попадает в жабры
			- С помощью мембраны может открываться и закрываться
		- Удобно, если рыба донная, т.к. в рот попадал бы песок
	- Поры
		- Поры с [[Электрорецепция рыб|ампулами Лоренцини]]
		- Поры эндолимфатических протоков
			- Это каналы, которые соединяют внешнюю среду и слуховую капсулу - отдел, содержащий внутреннее ухо
			- Открываются между брызгальцами
		- ![[Хрящевые рыбы-12-11-2025.png|300]]![[Хрящевые рыбы-16-11-2025.png|320]]
	- 5 жаберных щелей, разделенных септами
		- Без жаберных крышек
	- Клоака
		- У самцов с урогенетальным сосочком, у самок - с мочевыводным
	- У самцов есть класпер - удлинение брюшного плавника, для передачи семени
	- Плакоидная чешуя
		- ![[Хрящевые рыбы-16-11-2025 1.png|217]]
	- Хвостовой плавник гетероцеркальный
- Внутреннее строение
	- Спиральный клапан в кишечнике
	- Ректальная железа
		- Забирает из крови излишки солей и выделяет их в прямую кишку
		- Нужна, т.к. хрящевые живут в соленых водоемах
	- Нет плавательного пузыря
- Развитие
	- Всегда внутреннее оплодотворение
	- Может быть живорождение, яйцекладка, яйцеживорождение
	- Яйца - “кошельки русалки”
	- [яйцо акулы с живым эмбрионом внутри](https://vk.com/im?sel=262795950&z=video-37948240_456244423%2Fa579e66926a9097a6f%2Fpl_post_-37948240_1191691) и [Рожающий скат](https://vk.com/clip-37948240_456245464)
