# Избирательная система
Up:
Next:

- Состоит из
	- [[Избирательное право]]
	- [[Избирательный процесс]]
- **Избирательная система** - совокупность норм права, регулирующих общественные отношения в ходе избирательного процесса, а также деятельность субъектов политики в ходе организации, проведения и подведения итогов выборов
- Виды
	- [[Мажоритарная избирательная система]]
	- [[Пропорциональная избирательная система]]
	- Смешанная
		- По ней избирается [[государственная дума]], т.к. половина мест - по мажоритарной, а половина - по пропорциональной
