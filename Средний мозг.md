# Средний мозг

- Отдел мозга, часть [[ствол мозга]]
- Отвечает за реализацию многих физиологических функций
	- Зрение, слух, контроль движения
	- Цикл [[Сон]] и бодрствования
	- Общий уровень возбуждения ЦНС (схоже по ф-ции с [[Глутамат]])
	- Концентрация [[Внимание]]
	- Ориентировочные и защитные [[рефлекс]]ы
	- Регуляция [[боль]] чувствительности
	- [[Половое поведение]]
- В среднем мозгу располагается **четверохолмие**: два **зрительных** и два **слуховых** бугорка.
- Состоит из
	- [[Ретикулярная формация]] среднего мозга
	- [[Вентральная область покрышки]]
	- [[Черная субстанция]]
