# Форма обучения “погружение”

- На протяжении какого-то времени(2-3 недели) изучается только один предмет
- Опирается на [[Учение о доминанте]]
	- [[Классно-урочная форма обучения]] постоянно сбивает доминанту у учеников
	- Для качественного освоения ее нужно поддерживать длительное время
- Во время погружения допустимы уроки, не сбивающие доминанту(физкультура, изо, музыка)
- Домашнее задание обычно не задается
- [[Вопросы для изучения]]:
	- Что эффективней: сохранение доминанты или использование эффекта [[Искусственная актуализация знания]]?
