# Ассоциативное обучение

Является частью [[Формирование долговременной памяти]]. Работает так:
1. С помощью [[Глутамат]] белки-рецепторы синтезируются ВтП
2. Так же у данного [[Нейрон]]а есть синапс с нейроном [[Положительное подкрепление]]
Чтобы ассоциотивное обучение произошло, нужно чтобы эти два синапса сработали **одновременно**, тогда их ВтП повлияют на ДНК, та в свою очередь синтезирует новые рецепторы [[Глутамат]], которые встают на постсинапт. мембрану, усилив действие глутамата.
