# Прибыль
Up:
Next:

- **Бухгалтерская прибыль** - разница между общей выручкой и бухгалтерскими издержками
	- Бух. издержки (явные, внутренние) - платежи внешним по отношению к данной фирме поставщикам ресурсов (трудовым, материальным и тд)
		- То есть это любые затраты на производство
		- Аренда, проценты по [[кредит]]у, [[Заработная плата]] сотрудникам, амортизационные отчисления, оплата ресурсов поставщиков
	- Этот метод нужно использовать для подсчета действительной величины [[Издержки производства]] и прибыли
- **Экономическая прибыль** - разница между общей выручкой фирмы и |экономическими издержками
	- Экономические издержки = явные издержки + [[Издержки производства|альтернативные издержки]] (неявные)
	- Используется, когда нужно выбрать между несколькими альтернативными вариантами вложения ресурсов
- Рентабельность = прибыль/затраты
	- Показатель, использующийся для оценки эффективности предприятия
