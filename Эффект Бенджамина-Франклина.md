# Эффект Бенджамина-Франклина

- If a person helped us once, they are more likely to help us next time
	- It can be explained by [[Когнитивный диссонанс]] theory
		- If i help someone, then this person not so bad, because “would i help a bad person?”
		- Если человек делает кому-то добро, этот факт будет диссонировать с теми негативными ощущениями, которые он испытывает по отношению к нему
		- Если для совершения добра не будет достаточного оправдания ([[Высокая награда как способ решения когнитивного диссонанса, возникающего, когда человек занимается не тем, чем хочет]]), человек изменит свои аттитьюды по отношению к человеку для разрешения диссонанса
			- [[Негативные эмоции]] от человека уменьшаются
- By asking for help you show your admiration of persons skills - [[Способы понравится человеку]]
- Тот, кому сделали добро, захочет помочь в ответ вследствие [[Правило взаимного обмена]]
- Возможно, этим можно объяснить золотое правило морали - [[Моральный долг]]
