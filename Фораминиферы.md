# Фораминиферы
Up:
Next:

т. Foraminifera

- Обитают в морях
- Раковины - наружный скелет
- Типы раковин фораминифер
	- Органические
		- Состоят из псевдохитина
	- Инкрустированные
		- Из песчинок
	- Известковые
		- Самые распространенные
- Отмершие фораминиферы образуют мел, известняк
- Жизненный цикл
	- У большинства метагенез - чередование в цикле полового и бесполого размножения

- Систематика
	- кл. Allogromiida
		- Однокамерная раковина из псевдохитина
			- Из нее растут ритикулоподии
	- кл. Textulariida
		- Камеры распологаются в два ряда
		- ![[Фораминиферы-01-10-2024.png|300]]
	- кл. Fusulinida
		- Вымершие
		- Крупные раковины 20-30см
			- Веретеновидные известковые
	- кл. Miliolida
		- Известковая раковина
			- В виде беспорядочно расположенных камер с толстыми стенками
	- кл. Rotaliida
		- Раковины по спирали, более тонкие
		- ![[Фораминиферы-01-10-2024 1.png|300]]
