# Митоз растительной клетки
Up: [[Митоз]]
Next:

- Формируется фрагмопласт – цилиндр из коротких микротрубочек и микрофиламентов 
	- Фрагмопласт образуется на том месте, где было препрофазное кольцо
- Вдоль микротрубочек фрагмопласта начинают мигрировать пузырьки [[Аппарат гольджи]] с пектином
	- Эти пузырьки объединяются, формируя межклеточную пластинку
	- Разделяет клетку изнутри наружу
- Затем по обе стороны от межклеточной пластинки клетки достраивают слой из целлюлозы, формируя клеточную стенку
	- Таким образом, клетки растения оказываются вплотную друг к другу, а между ними – межклеточная пластинка
	- ![[Митоз-03-08-2025 1.png|611]]
- ![[Митоз-03-08-2025.png|802]]

- Так как для деления клетки по центру должен образоваться фрагиопласт, а потом и межкл. пластинка, обычная клетка делиться не может - ее центр уже занят огромной вакуолью
	- поэтому есть специализированные для деления клетки - [[Образовательная ткань растения]]
