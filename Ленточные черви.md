# Ленточные черви
Класс Cestoda
![[Ленточные черви-31-08-2024.png|479]]

- Развитие со сменой хозяина - [[Смена хозяинов в жизненном цикле паразита]]
	- Промежуточный хозяин - крупный рогатый скот
	- Окончательный хозяин - человек
- Взрослые особи живут в [[кишечник]] [[Позвоночные]]
- Строение
	- Ни на одной стадии нет [[Пищеварительная система]]
		- [[Общая дегенерация]] из за паразитического образа жизни
	- Также как и у [[Трематоды]] есть тегумент (неодерма), помогающий всасывать питательные вещества всей поверхностью
- Строение тела
	- Сколекс (головка)
		- Присоски
		- Крючья
			- Обычно крючья образуют венчик
		- Ботрии
			- Похожа на прищепку (из двух створок с щелью)
			- С помощью нее прицепляется к стенке эпителия кишечника хозяина
			- ![[Ленточные черви-25-11-2024 1.png|433]]
		- Ботридии
			- Сложные разветвленные присоски
			- ![[Ленточные черви-25-11-2024.png|246]]
	- Шейка
		- Содержит большое количество [[стволовые клетки]]
		- Здесь образуются новые проглоттиды
	- Стробила (тело)
		- Состоит из члеников - проглоттид
			- Тело червя может содержать от 3 до 4000 проглоттид
	- ![[Свиной и бычий цепни.excalidraw]]
- Системы органов
	- Протонефридиальная система
		- 4 продольных канала (2 спереди, 2 сзади по бокам)
		- Продольные каналы соединены друг с другом поперечными
	- Нервная система - ортогон
		- Схожа по строению с протонефридиальной
	- Тегумент
		- На поверхности синцитиального слоя есть выросты - микротрихии (1)
		- ![[Ленточные черви-25-11-2024 2.png|276]]
	- ![[Половая система цестод]]
- Финна
	- Покоящаяся стадия
	- Сферическая форма
	- Содержит один или несколько ввернутых внутрь сколексов
	- Нужна для ослабления хозяина
- Методы профилактики
	- Качественная прожарка мяса
- Классификация п/кл Eucestoda
	- о. Pseudophyllidea (псевдофиллиды)
		- Присосок нет, только ботрии
		- Представители
			- [[Широкий лентец]]
			- Ремнец - Ligula intestinalis
				- У него нет разделения на проглоттиды
				- Основной хозяин - рыбоядные [[птицы]]
					- Плероцеркоиды в рыбе меняют ее поведение, заставляя плавать на поверхности брюхом кверху
					- Так птица с большей вероятностью заметит и съест рыбу
	- о. Cyclophillida (циклофиллид)
		- Не бываает ботрий, есть присоски или крючья
		- Представители
			- [[Свиной цепень]]
			- [[Бычий цепень]]
			- [[Овечий мозговик]]
			- [[Эхинококк]]
- Представители п/кл Eucestoda
	- Бычий цепень, свиной цепень
		- Свиной цепень имеет крючья на головке
	- Альвеококк
