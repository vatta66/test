# Амебы
Тип Aamoebozoa

[[Системы для урока Простейшие.canvas|Системы для урока Простейшие]]

## Для вуза

- [[Псевдоподии]] типа лобоподий

- [[Полифилетический таксон]]
- Отсутствие жгутиков
- Отсутствие клеточного центра (центриолей)
- Движение ложноножками (псевдоподиями)
	- Для этого вся клетка пронизана актином и миозином
- Сократительная вакуоль
	- Для избавления от избытка воды и продуктов метаболизма
- При неблагоприятных условиях инцистируются
	- Превращаются в цисту - стадия, на которой амеба переживает неблагоприятные условия 
- Амебоидное движения
	- Появляется ложноножка, затем все содержимое клетки перетекает в нее и так далее
	- ![[Амебы-30-08-2024.png|300]]
- Может охотиться на [[Хламидомонада]]
- Размножение
	- [[Бесполое размножения]]
		- Бинарное деление
		- Шизогония
		- Плазмотомия
	- Парасексуальный процесс
- Классификация
	- п/т Lobosoa
		- Класс Gymnamoebea (голые амебы)
		- Класс  Acarpomyxea
			- Сильно ветвящиеся псевдоподии (ретикулоподии)
		- Класс Testacealobosea (лобозные раковинные амебы, тестации)
			- Тело частично покрыто раковиной
				- Похожая есть у [[Фораминиферы]]
	- п/т Conosa
		- класс Archamoeba (архамебы)
			- Утратили [[митохондрия|митохондрии]] и диктосомы, слабо развит [[Эндоплазматическая сеть|шероховатый ЭПС]]
			- Сюда относится Entamoeba histolytica (дезентерийная) и Entamoeba coli
- Разнообразие амеб
	- Акантамеба
		- Факультативный паразит
			- Может жить в водоеме, а может быть паразитом
		- Вызывает акантамебный кератит - амеба поедает [[Строение глаза|роговицу глаза]] человека
	- Arcella gandalfi
		- У этой амебы есть раковина из [[хитин]]а, в центре которой находится устьице (из них высовываются ложноножки)
		- Названа так в честь шляпы гендальфа
		- ![[Амебы-30-08-2024 1.png|300]]![[Амебы-30-08-2024 2.png|300]]
	- Ротовая амеба
		- Живет в [[зубы|зубах]] с кариезом
		- Вреда не причиняет, питается [[бактерии|бактериями]]
	- Миксомицеты
		- Из споры мискомицета выходит гаплоидная клетка
			- Если сухо - гаплоидная амеба
			- Если влажно - гаплоидный жгутиконосец
		- Эти амебы живут, размножаются с образованием таких же гаплоидных клеток
		- Далее при размножении эти гаплоидные клетки сливаются и образуют диплоидную зиготу (тоже амеба)
		- Далее эта диплоидная амеба продолжает питаться, но не делиться
			- Делятся только ее ядра (могут достигать миллионов штук)
	- Амебы, живущие в кишечнике человеке
		- Entamoeba coli - [[Комменсализм|комменсал]] человека, не причиняет вреда
			- ! У их цист 8 ядер
		- Entamoeba histolytica - дизентерийная амеба
			- [[Паразитизм|Паразит]], вызывает кишечний амебиаз
			- Вторая по количеству смертей в год причина смерти от [[Паразитизм|паразитов]]
			- ! У их цист 4 ядра
- Интересные факты
	- Геном амёбы Amoeba dubia состоит из 690 млрд пар нуклеотидов, а у человека  - всего 2.9 млрд
