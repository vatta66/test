# хорда
Up: [[хордовые]]
Next:

![[хорда-02-09-2025.png|465]]

- Гибкий упругий тяж, развивающийся из [[мезодерма]] (по последним данным)
- Есть белковая оболочка, за ней коллагеновая, а внутри – клетки с большими вакуолями с жидкостью
	- Такое строение позволяет не сжимать само тело при сокращении мышц
	- Движение происходит “вилянием” из стороны в сторону - ундулирующий механизм
- Развивается из гензеновского узелка
- хорда может сохраняться на взрослой стадии в виде межпозвоночных дисков
- хорда самая длинная из трех трубок
