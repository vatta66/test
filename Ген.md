# Ген

- Любая порция хромосомного материала, сохраняющаяся на протяжении большого числа поколений
-  Ген бессметрен
	- Существует в виде своих копий
- Аллели гена - его соперники, а другие гены - среда
	- Отбор идет по тем генам, способным эффективно взаимодействовать с другими генами
-  Не контролируют действия своей машины для выживания
	- Они заранее дают им инструкции, переклаывая управление на мозг
- Ничто не хочет эволюционировать
	- Гены используют тело, чтобы оставаться неизменными
