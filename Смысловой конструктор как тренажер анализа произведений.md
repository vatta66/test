# Смысловой конструктор как тренажер анализа произведений

- Выписываются все значимые детали произведения (цитатами), разносятся по заметкам
	- Описания, действия, речевые обороты и тд
- Все это линкуется в группы
	- Это и есть конструктор
- Каждый ученик должен будет “собрать” этот конструктор, делая связи, создавая новые заметки своих мыслей
- Такая практика будет отличной тренировкой анализа произведения
- Будет идти развитие [[Навык системного мышления]], умение находить связи, анализировать
