# Проводяще пучки
Up: [[Проводящая ткань растения]]
Next:

- Простые пучки
- Сложные пучки
	- Открытые
		- *Камбий есть*
	- Закрытые
		- *Камбия нет*

- По расположению ксилемы и флоэмы
	- Коллатеральные
		- Тяжи ксилемы примыкают к тяжам флоэмы
		- Характерно для многих семенных
	- Биколлатеральнеы
		- Тяжи ксилемы между двумя тяжами флоэмы
		- Характерно для некоторых двудольнх
	- Радиальные
		- Радиумы ксилемы, между ними большие пучки флоэмы
		- Характерно для [[Корень растения|кореня]]
	- Концентрические
		- Амфикрибральные - флоэма вокруг ксилемы
			- В основном у [[Папоротниковидные]]
		- Амфивазальные - ксилема вокруг флоэмы
			- Есть у некоторых [[Однодольные растения]]
