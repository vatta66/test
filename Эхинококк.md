# Эхинококк
Up: [[Ленточные черви]]

Echinococcus granulosus
- Очень маленькие - 3 проглоттиды ![[Эхинококк-02-07-2025.png|214]]

- Основной хозяин - псовые
	- Прикрепляются к микроворсинкам кишечника, т.к. очень маленькие
	- В проглоттидах созревают яйца
	- Проглоттиды не отсоединяются, а имеют выводные протоки для яиц
- Яйца глотают [[млекопитающие]]
	- Онкосфера сбрасывает ресничный покров
	- Попадает в кровь
- Переходит в любой орган
- Там образуется финна
	- Похожа на ценур - есть несколько сколексов
- В отличие от ценура может образовывать дочерние финны
	- Получается матрешка из финн - эхинококковый пузырь
	- Может достигать огромных размеров
- Финну съедает основной хозяин
