# Эффект Зейгарник

- Незавершенные задачи запоминаются лучше (1,2)
- Ваше [[внимание]] будет сконцентрировано на незавершенной задаче
	- Это будет вызывать эффект [[Повышение важности идеи при направлении на нее внимания]]
	- Вы будете думать, как могли бы завершить эту задачу
- Эффект становится сильнее, если 
	- Мы недосказываем важный результат
	- Человек считает себя обязанным выполнить что-то
	- Человек далеко продвинулся по выполнению задачи
	- У человека есть [[Внутренняя мотивация]] узнать что-то/сделать что-то
- Применение 
	- На школьниках - [[Педагогика MOC]]
		- Например, можно начинать урок с тайны, к разгадке которой они будут идти весь урок, а затем, используя новые знания, разгадают ее
	- Можно увеличивать [[Внутренняя мотивация]] работать и начинать работать
		- Это можно рассматривать как еще одно решение проблемы [[Оттягивания начала поведения с долговременной программой]]
- Примеры
	- Лучше всего запоминались подробности рекламы, которая была прервана за 5-6 секунд, по сравнению с завершенной рекламой (3) - [[маркетинг]]
		- Эффект становился отчетливей спустя два дня, две недели (увеличивалась разница между двумя рекламами)
