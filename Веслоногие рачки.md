# Веслоногие рачки

отряд из подкласса [[Максиллоподы]] класса [[Ракообразные]]

- Пресноводные, маленькие (несколько мм)
- Много паразитических особей с приспособлениями
	- ![[Максиллоподы-31-03-2025 1.png|648]]
	- ![[Максиллоподы-31-03-2025 2.png|384]] - эта самка из копепод мимикрирует под яйца высших раков, поэтому они заботятся о ней как о собственном потомстве
- Подотряд Cyclopoida
	- Представитель - Cyclops (циклоп)
		- ![[Максиллоподы-31-03-2025.png|434]]
		- Цефалоторакс
			- 1 науплиальный глазз
			- Длинные 1 антенны, 2 - короткие
			- Двудетвистые мандибулы
		- Торакс из 3 сегментов
			- Двуветвистые периоподы
		- Цефалоторакс и тельсон называют **просомой**
		- Два генитальных сегмента
		- Абдомен из 3 сегментов
			- Без конечностей
		- Генитальные сегменты + абдомен - уросома
		- Тельсон с членистой фуркой
		- ![[Веслоногие рачки-31-03-2025.png|300]]
- Копепода Neoalbionella globosa может поселяться в [[Обоняние рыб|обонятельной системе рыб]]
