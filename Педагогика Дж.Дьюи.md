# Педагогика [[Дж.Дьюи]]

- Концепция полного акта мышления
	- Человек начинает мыслить, когда сталкивается с трудностями и проблемами, преодоление которых имеет для него важное значение
	- Проблемы отличаются от [[Педагогика И.Ф.Гербарта|гербартистских]], мнимых проблем, не волнующих и не интересующих ученика
		- По Дьюи, учитель должен следить за развитием и интересами учащихся, и на основе этого предлагать новые проблемы
			- Можно использовать в [[Проблемное обучение]]
- [[Прагматизм в педагогике]]
- Место учителя
	- Ничего не навязывает
	- Лишь помогает преодолевать трудности, когда обучающиеся его об этом просят
- Минусы
	- Учащиеся не участвуют в процессе закрепления знаний, развития умений
	- Отсутствие преемственности систематичности в обучении
		- Преемственность - установления связи между частями предмета на разных стадиях его изучения
