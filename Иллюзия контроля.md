# Иллюзия контроля

- One of the [[Когнитивные искажения]]
- People prone to think that they can control much more that they really can
	- Maybe it can cause a [[Ошибка планирования]]
- Factors influence the control illusion [1]
	-  Personal involvement
	- Extent of familiarity
		- In a new context this distortion less likely to occur
		- Maybe because of [[Когнитивная легкость]] created by familiar context
	- [[Негативные эмоции]] decrease this effect occurrence
		- Maybe because [[Когнитивная легкость]] doesnt activated
- [[Не стоит полагаться на удачу]]
