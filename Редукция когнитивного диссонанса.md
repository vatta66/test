# Редукция когнитивного диссонанса

- Когда человек уже осуществил выбор, положительные стороны невыбранных вариантов диссонируют с фактом их невыбора, а негативные стороны выбранного - с фактом их выбора - появляется [[Когнитивный диссонанс]]
- Для его решения человек будет умалять невыбранные варианты, а выбранный будет еще сильнее подниматься в его глазах
	- Похоже на [[Изменение восприятия человека при влюбленности]]
- Особенно сильно этот принцип наблюдается при выборе из “двух зол”
- Мои гипотезы
	- Если человек выбирает между двумя действиями, то после выбора, вследствие редукции диссонанса, у него появится [[Внутренняя мотивация]] для совершения этого действия
