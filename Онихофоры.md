# Онихофоры
Тип Onychophora

- Наземные
- Живут во влажных тропических лесах
- Сегмент
	- По паре лопостевидных конечностей
		- Не членистые
		- С когтями
	- По паре [[Метанефридии]]
		- В основании ног
- У [[Кутикула членистоногих|эпикутикулы]] нет водоупорной функции
	- В связи с этим они могут жить только
- [[Мускулатура червеобразных]]
	- Т.к. передвигается тоже червеобразным способом
	- Мускулатура ног
- Голова
	- Протоцеребрум
		- Иннервирует антеныы и глаза
	- Дейтоцеребрум
		- Иннервирует челюсти
			- Для вскрывания покровов жертвы
	- Подглоточный ганглий иннервирует оральные папиллы
	- Есть рога с клейкой жидкостью
		- Служит для защиты
- Охота
	- С помощью челюстей вскрывает покровы тела жертвы
	- Впрыскивает в нее секрет [[слюна|слюнных]] желез
		- Слюнные железы - видоизмененные [[Метанефридии]]
			- Выделяют пищеварительные ферменты
		- Всасывают переваренную пищу
			- Внешнее пищеварение
	- Оральные папилы
		- В них открываются протоки слизевых желез
			- Склеивает жертв
				- В основном - [[Насекомые]]
			- За один выстрел расходует всю слизь
- [[Трахейная дыхательная система]]
	- Устроена примитивнее, чем у настоящих членистоногих
		- Больше дыхалец
		- Дыхальца не закрываются
- Размеры - от нескольких до десятков см.
