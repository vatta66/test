# Декомпозиция цели для получения подкрепления

- При [[Целеполагание]] вы должны разделять большую цель на маленькие ([[декомпозиция]]), чтобы получать [[Положительное подкрепление]] после выполнения каждой части
- Это может послужить решением проблемы [[Оттягивания начала поведения с долговременной программой]]
