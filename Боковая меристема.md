# Боковая меристема
Up: [[Образовательная ткань растения]]
Next:

- Вторичны по происхождению
- Нужны для утолщения осевых органов
- Виды
	- Камбий
		- Два типа клеток
			- Веретеновидные инициали
				- Вытянутые, вакуализированные
				- В стебле они всегда дифферинцируются из прокамбия, в корне - из плеромы
				- Из них образуются проводящие и механические элементы флоэмы и ксилемы, а также иногда паренхимные клетки
			- Лучевые инициали
				- Дифферинцируются из паренхимы корня, либо из центрального цилиндра, либо при делении веретеновидных клеток
				- Из них дифферинцируются паренхимные клетки
	- [[Феллоген]]
		- Обеспечивает возникновение вторичной покровной ткани
		- Закладывается по периферии многолетних органов из разных тканей (из эпидермы, гиподермы, флоэмы)
			- У корня всегда появляется из перицикла
		- Функция
			- Делясь наружу откладывают [[пробка|пробку]] (феллема)
			- Делясь внутрь образуют феллодерму
				- Живую паренхимную ткань
			- Вместе феллема + феллоген + феллодерма  - перидерма
