# Создание [[Классификация]]

1. Применить к множеству объектов [[Выделение главного]]
2. Взять один объект и поочередно осуществлять [[Сравнение]] других элементов с элементами его множества
	1. Можно создать [[Моделирование|модельный объект]] данного множества и сравнивать новые элементы только с ним
	2. В случае совпадения добавить элемент в множество
3. Отвергнутый объект взять за основу другого множества
4. Сравнивать оставшиеся объекты с существующими множествами и повторять п. 2 и 3

- Нужно для создания [[Классификационное знание]]
