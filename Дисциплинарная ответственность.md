# Дисциплинарная ответственность
Up: [[Юридическая ответственность]] - [[Педагогика MOC]]
Next:

- За дисциплинарные проступки
	- Неисполнение [[Трудовое право|работником]] или [[образование|учеником]] своих обязанностей
- С 14 лет
- Меры обеспечения
	- Доставление
		- Принудительное препровождение
	- Задержание
		- Кратковременное лишение свободы
	- Личный досмотр, досмотр вещей и тд
		- Только в присутствии 2 понятых
- Виды наказания
	- Выговор
	- Замечание
	- [[Расторжение трудового договора#^90a9dd|Увольнение]] (исключение из учебного заведения)
- Для несовершеннолетних ^4694fd
	- Для работающих
		- Также за неисполнение правил внутреннего распорядка
		- Замечание, выговор, увольнение
	- Для учащихся
		- За неисполнение устава ОУ, правил внутреннего распорядка
		- Виды наказаний
			- Замечание, выговор
			- Отчисление
				- Применяется только с 15 лет
- Не несут дисциплинарной ответственности
	- Учащиеся детских дошкольных образовательных учреждений (ДДОУ)
	- Болеющие ученики
	- Дети начальной школы
	- Дети с ограниченными возможностями
