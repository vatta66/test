# Extended systemic recall - Свяжи и передай
Up: [[Systemic recall]]
Next:

- Идея со скиллбокса [1], адаптированная под [[теория системного обучения]]
- В исходной идее была проблема - записать одно [[Атомарное знание]] будет слишком мало, это будет очень неэффективная трата времени
- Данный метод можно применять после прохождения темы
	- Ученики своими руками будут создавать [[смысл]] знаний, связывая их
	- Смысл будет постепенно появляться прямо у них на глазах
- Шаги
	- Раздайте каждому ученику по листу чистой бумаги
	- Учитель засекает определенное время (сколько необходимо?), за это время ученики записывают один небольшой [[Кластер знаний]], который они усвоили с урока
		- Он может состоять всего из 3 атомарных знаний: митохондрии раньше были бактериями - митохондрии вырабатывают энергию - энергия нужна для того, чтобы клетка выполняла свои функции (синтез белка и тд)
		- Количество знаний в кластере неограничено, каждый имеет право создать кластер любой величины, главное чтобы в нем не было не связанных знаний
	- Листки собираются и раздаются в случайном порядке
	- Каждый ученик должен написать еще один мини кластер, который свяжется с предыдущим
		- При том нельзя писать то, что ты уже писал
		- В клетке есть ядро - ядро имеет такие то части - функция ядра-хранение ген инф, а она нужна для синтеза белка, для которого нужна энергия, производимая митохондрией
	- Проведите несколько таких раундов, после чего предложите каждому найти свой исходный листок.
		- Проведите обсуждение: попросите учеников порефлексировать о записях, которые они видят в своём листе. Например, назвать самые интересные идеи, которые записали туда другие.
- Эти шаги можно повторять столько раз, сколько пожелает учитель
	- После определенного количества итераций ученикам ничего не останется, кроме как вводить связи со знаниями с прошлых уроков
		- Это чем то напоминает [[Принцип доступности связи при создании нового витка спирали|принцип доступности связи]]
- Используется практика извлечения (retrieval)
	- Является основой для [[Интервальное повторение]]
- ? Подумать, как можно подвязать сюда [[Виды связей в теории системного обучения]]


Каждому ученику выдается лист бумаги, на котором за определенное время они должны записать небольшой кластер знаний по теме. Он может состоять всего из 3 атомарных знаний. Например, митохондрии (специальные части клетки) раньше были бактериями - митохондрии вырабатывают энергию - энергия нужна для того, чтобы клетка выполняла свои функции, такие как синтез белка и др. Листы собираются и распределяются случайным образом между учащимися. Каждый ученик должен добавить еще один мини-кластер, связанный с предыдущим, но не повторяющий знания, уже записанные на листе. Например, в клетке есть ядро - ядро имеет такие-то части - функция ядра-хранение ДНК, а она нужна для синтеза белка, для которого нужна энергия, производимая митохондрией. Эти шаги могут повторяться несколько раз по желанию учителя. В конце можно вернуть ученику его листок, чтобы он посмотрел, какие знания написали его одноклассники и как связали их с его знаниями. Если позволяет время, можно провести рефлексию над этими работами, а если нет - попросить учеников написать, какие самые интересные связи и знания написали на его листок другие.
