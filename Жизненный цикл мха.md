# Жизненный цикл мха
Up:
Next:

- Гаметофит состоит из 2 стадий:
	- Протонема (молодая)
	- Гаметофор (взрослая)

- Гаметофор
	- Есть деление на мужские и женские гаметы
- Растет [[митоз]]ом
- Мужские гаметы оплодотворяют женские
- На женском гаметофите развивается спорофит 2n (коробочка на ножке)
	- В ней из спорогенной ткани (2n) образуются споры (n) с помощью [[Мейоз]]
- Споры высыпаются на землю, проростают, образуя протонему (предросток)
	- Однорядная нить с ответвлениями
	- Протонема растет [[митоз]]ом, вскоре на ней образуются почкообразные структуры
- Из почек протонемы вырастает гаметофор
