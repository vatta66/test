# Качественный анализ
Up: [[Аналитическая химия]]
Next:

- Определяет состав вещества
- В нем происходят легко выполнимые реакции (легкие условия)
- Строится на ионных реакциях в водных растворах 
- Методы качественного анализа
	- Пробирочный
		- Реактивы смешивают в пробирках-центрифуге
	- Микрокристаллоскопический
		- Помогает обнаружить кристаллы, которые окрашиваются определенным образом
	- Капельный
		- Можем обнаружит и разделить компоненты
		- Используется на спец. бумаге с капиллярами
- Аналитический реагент (реактив) - вещество, которое действует на открываемый компонент
- Аналитические реакции
	- Специфические
		- Можно обнаружить только данное вещество
	- Избирательные
		- Можно обнаружить небольшую группу веществ
- Свойства аналитических реакций
	- Избирательность - разные сигналы от разных реакций
		- Для повышения избирательности нужно соблюдать pH, температуру, концентрацию ионов, природу растворителя
- Мешающие ионы - мешают обнаружить искомое вещество
	- Проблемы
		- Либо образуют соединения, похожие на целевые
		- Либо их реакция с реагентом затмевает реакцию искомого вещества
		- Либо они связывают искомое вещество, и его просто не остается на реакцию с реагентом
	- Решение
		- Разделение - разделяем мешающие ионы и целевое вещество по двум разным фазам
		- Маскировка - проводим реакции, уменьшающие концентрацию мешающих ионов
- Классификации ионов
	- Катионы
		- ![[Качественный анализ-17-09-2025.png|300]]
	- Анионы
		- ![[Качественный анализ-17-09-2025 1.png|300]]
- Методы анализа
	- Систематический
		- Мы последовательно удаляем ионы из раствора, остается только один конкретный
	- Дробный метод
		- Мы не отделяем ионы друг от друга, а обнаруживаем их все в одном растворе
