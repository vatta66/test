# Майевтика

- Метод для извлечения истины из человека 
	- с помощью [[Вопросы]]
- Суть:
	- Вместо того, чтобы утверждать что-либо, задавать наводящие вопросы
	- Отвечая на эти вопросы, собеседник формулирует до этого неизвестные ему утверждения
- Используется в [[Метод Сократа]]
- Похоже на [[Эвристическая беседа]]
- Может служить хорошим приемом для 
	- [[Доминирование в переговорах]]
	- [[Уход от ответа]]
	- [[Манипуляции MOC]]
