### Закон действующих масс в аналитической химии

### Произведение растворимости

Для малорастворимого электролита общего вида  
AₘBₙ ⇄ mAⁿ⁺ + nBᵐ⁻  
в насыщенном растворе устанавливается равновесие между твердой фазой и ионами в растворе.

По закону действующих масс константа равновесия имеет вид:
K = [Aⁿ⁺]ᵐ · [Bᵐ⁻]ⁿ / [AₘBₙ]

Концентрация твердой фазы постоянна и включается в константу равновесия. Вводят величину **произведение растворимости**:

ПР = [Aⁿ⁺]ᵐ · [Bᵐ⁻]ⁿ

Произведение растворимости — постоянная величина для данной температуры и характеристика равновесия между осадком и насыщенным раствором.

#### Вычисление произведения растворимости по растворимости

Если растворимость соли равна S (моль/л), концентрации ионов выражают через S с учетом стехиометрии диссоциации.

Пример для соли AB:  
AB ⇄ A⁺ + B⁻  
[A⁺] = S, [B⁻] = S  
ПР = S²

Пример для соли A₂B:  
A₂B ⇄ 2A⁺ + B²⁻  
[A⁺] = 2S, [B²⁻] = S  
ПР = (2S)² · S = 4S³

### Дробное осаждение. Превращение труднорастворимых электролитов

Дробное осаждение — последовательное выпадение осадков при постепенном увеличении концентрации осаждающего иона.

Осаждение начинается при выполнении условия:
[катион]ᵐ · [анион]ⁿ > ПР

**Соль с меньшим значением ПР осаждается первой.**

Превращение одного труднорастворимого электролита в другой возможно, если ПР образующегося осадка меньше и ионы удаляются из раствора.

Реакции обмена протекают при образовании:
- осадка,
- газа,
- слабого электролита.

### Условия образования и растворения осадков. Полнота осаждения. Солевой эффект

Осадок образуется, если ионное произведение превышает ПР.

Осадок растворяется, если ионное произведение меньше ПР.

Полнота осаждения определяется величиной ПР, концентрацией реагентов и составом раствора.

Солевой эффект заключается в увеличении растворимости малорастворимых электролитов при добавлении инертных солей за счет снижения коэффициентов активности ионов.
