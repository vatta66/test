# Урок по теме Нуклеиновые кислоты

[[Системы для урока Нуклеиновые кислоты.canvas|Системы для урока Нуклеиновые кислоты]]

- Есть 2 вида нуклеиновых кислот: ДНК и РНК
	- Это полимеры, мономер которых - нуклеотид
- Нуклеотид состоит из
	- Азотистое основание
		- У ДНК
			- Аденин - Тимин
			- Гуанин - Цитозин
		- У РНК
			- Аденин - Урацил
			- Гуанин - Цитозин
		- _Принцип комплементарности_
		- Между азотистыми основаниями - водородные связи
	- Остаток фосфорной кислоты - фосфат
	- Остаток сахара
		- У ДНК - дезоксирибоза
		- У РНК - рибоза
