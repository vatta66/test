# Монархия
Up: [[Формы правления]]
Next: [[Республика]]

- **Монархия** - [[Формы правления|форма правления]], при которой верховная [[Государственная власть]] осуществляется единолично, не сменяется и передается по [[наследование|наследству]]
- Признаки
	- Бессрочность и несменяемость
	- Передача власти по наследству
	- Отсутствие юридической ответственности монарха за свои решения
- Абсолютная
	- Вся [[Государственная власть]] принадлежит монарху
	- Тем не менее, ветви власти есть
	- Примеры: Объединенные Арабские Эмираты, Саудовская Аравия, Катар
- Ограниченная (конституционная)
	- Дуалистическая
		- Власть разделяется между правительством и парламентом
			- [[Правительство]] формирует монарх
				- Соответственно, оно ответственно перед монархом
			- [[Парламент]] избирает народ из приближенной к монарху семьи
		- Монарх обладает правом вето на решения парламента
			- Также может распустить его
		- Может оказывать реальное влияние на внешнюю политику
		- Примеры: Марокко, Иордания, Непал
	- Парламентская
		- Монарх выполняет представительскую ф-цию
			- То есть его роль в политических решениях незначительна
		- Законодательная власть - парламент, исполнительная - правительство
			- Премьер-министр - лидер победившей на выборах [[Политическая партия]]
				- То есть институт [[Избирательный процесс|выборов]] есть
		- Монарх не может распустить парламент
		- Примеры: Великобритания, Нидерланды, Дания
		- По сути похож на парламентскую республику
