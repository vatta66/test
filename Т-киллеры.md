# Т-киллеры
CD8+ хелперы

- Проверяет [[непрофессиональные антигенпрезентирующие клетки]]
- Имеет белок CD8
	- Нужен для более точного распознавания
	- Т-киллер взаимодействует с [[MHC1]] с помощью белка CD8
- Если клетка оказывается зараженной, [[рак]]овой и тд., т-киллер уничтожает ее внешним путем [[апоптоз]]а
- Из-за наличия cd4 рецептора подвержены заражению [[вич]]
