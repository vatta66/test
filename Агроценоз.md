# Агроценоз

- Искусственное сообщество ([[Экосистема]]), созданное человеком для получения сельскохозяйственной продукции
	- Агроэкосистема — искусственная экосистема, включающая агроценоз
- Примеры агроценоза
	- Сады, поля, огороды, пруды, плантации, виноградники, палисадники, аллеи, парки, пастбища
- Свойства агроценоза
	- Незамкнутый [[Круговорот веществ]]
		- Человек постоянно вмешивается в него, забирая себе часть продукции экосистемы
	- Слабая саморегуляция
	- Высокая численность одного вида (монокультура)
	- Невысокое видовое разнообразие
	- Короткие [[Пищевая цепь|цепи питания]]
	- [[Искусственный отбор]] + [[естественный отбор]] — сохраняются особи с нужными человеку признаками
	- Используются доп источники энергии
		- Например, [[Агротехнические приемы|удобрения]]
