# При глубоком погружении в тему она становится интереснее

- Когда вы начинаете глубоко изучать какую-то, возможно, прежде для вас не интересующую, тему, она становится для вас намного интереснее
- Это происходит потому, что при глубоком изучении любой темы (например при подоходе [[Фреймворк изучения темы]]) у вас формируется [[Кластер знаний]]
	- Вследствие чего начинает работать [[Искусственная актуализация знания]]
- Схожие идеи:
	- Может помочь в реализации [[Потребность в системном знании]]
		- Т.к. интерес будет являться [[Положительное подкрепление]]
- Что из этого следует/как это использовать:
	- Большая [[Внутренняя мотивация]] учеников при внедрении [[Мой проект по внедрению системы цеттелькастен в школы]]
- На основе этой идеи выведен [[Принцип глубины погружения в системной спирали изложения]]
