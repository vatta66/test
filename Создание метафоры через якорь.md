# Создание метафоры через якорь

- Создать эмоциональную ассоциацию с метафорой можно с помощью [[якорь в психологии]]
- Метафора может стать триггером, вызывающим определенное состояние
	- Т.е. как [[Воздействие образов, ассоциированных с определенным состоянием]]
- Возможно, особенно яркие моменты жизни еще лучше закрепятся при применении метафоры
	- Это поможет в [[Эмоциональная окраска этапов жизни]]
	- Метафора начинает ассоциироваться с данным эмоциональным состоянием - становится якорем
	- Метафора будет помогать - [[Позитивные эффекты метафор]]
- Получается создание [[Прайминг]] с помощью якоря
