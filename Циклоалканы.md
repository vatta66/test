# Циклоалканы
Up:
Next:

- Виды
	- Малые - 3,4 атома
	- Средние - 5, 6 атомов
	- Большие

Циклические углеводороды состава CnH2n, углероды связаны одинарными связями. Для малых циклов(циклопропан, циклобутан) характерны реакции присоединения, для больших циклов(циклопентан, циклогексан) - замещения
	
### Методы получения циклоалканов
1. Гидрирование аренов (+H2)
2. Дегалогенирование дигалогеналканов - циклизация(+Zn, Mg, Na, Li). Галогены должны находится не у соседних атомов углерода - чаще всего для малых циклов
3. Дегидроциклизация алканов(+H2). У алкана отщепляются концевые атомы водорода, алкан замыкается в цикл.
4. Гидрирование циклоалкенов

### Хим св-ва циклоалканов
Реакции **присоединения** для малых циклов:
1. Гидрирование(+H2) kat: Pd, Pl, Ni. В цикле происходит разрыв какой-то из связей, и к каждому из углеродов прибавляется водород, образуя алкан
2. Галогенирование. Происходит разрыв какой-то связи, и к каждому углероду присоединяется галоген. Важно, чтобы галоген присоединялся не к соседним углеродам
3. Гидрогалогенирование(HCl, HBr). Также будет разрываться любая связь(при условии отсутствия заместителей), к одному углероду будет присоединятся углерод, к другому - водород
	Если есть какие либо заместители, то связь будет разрываться около него(любая). Дальше водород присоединится к более гидрированному углероду, галоген - к оставшемуся
	
Реакции **замещения** для больших циклов:
1. Галогенирование. Любой из атомов водорода замещается на атом галогена, побочный продукт: кислота
2. Нитрование(+HNO3). Нитрогруппа(NO2) замещает любой водород, а замещенный водород образует воду с OH
