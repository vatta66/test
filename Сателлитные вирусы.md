# Сателлитные вирусы
Up: [[Вирусы]]
Next:

- Вирусы, которые для своей репликации полагаются на других вирусов
- В их геноме кодируется только один белок
- Они очень похожи на [[вироиды]]
	- Поэтому есть мнение, что они произошли именно от них в ходе [[Горизонтальный перенос генов|горизонтального переноса]] вироидов от растений к животным
- Например, вирус-сателлит некроза табака - он кодирует только белки своего капсида, и их ему помогает сделать сам вирус некроза табака
- Другой пример - спутник вирофаг
	- Он паразитирует на [[мимивирусы]] - он использует их аппараты репликации и транскрипции, то есть по сути является паразитом для этого вируса
