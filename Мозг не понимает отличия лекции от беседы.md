# Отличие [[лекция]] от беседы
Up: [[Техники эффективного обучения]]
Next:

- У беседы нет четкого плана, у лекции он есть
	- Из за этого в беседе мысль, сказанная в данный момент, скорее всего будет связана с мыслью, сказанной моментом ранее
	- А в лекции, благодаря плану, лектор может связывать идеи, находящиеся очень далеко друг от друга
- Ваш мозг привык слушать беседы, т.к. он делает это чаще
- Поэтому он будет воспринимать лекцию так же, как и ее, что неправильно
	- Для решения этой проблемы нужно вести системные конспекты
- Что из этого следует
	- Для преподавателей - [[Нужно проговаривать связи по ходу лекции (урока)]]
	- Для учеников - [[Вопросы, которые нужно задавать себе по ходу лекции]]
