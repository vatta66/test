# ДНК-полимераза
Up: [[Репликация ДНК]]
Next:

- [[Исправление ошибок при репликации]]
- Когда ДНК-полимераза садится на ДНК, она проедет только несколько десятков нуклеотидов и отцепится
	- Для того, чтобы этого не происходило, существует фермент PCNA - он удерживает полимеразу на ДНК
		- Выглядит как кольцо, которое едет по цепи ДНК вместе с полимеразой
	- Процесс установки PCNA
		- Спец. фермент “загрузчик” прикрепляется к закрытому PCNA, тратит 1 АТФ на то, чтобы немного раскрыть его (размыкает кольцо)
		- В таком состоянии PCNA может сесть на цепь ДНК
		- Загрузчик с затратой 1 АТФ обратно замыкает PCNA в кольцо и отцепляется
		- ![[ДНК-полимераза-22-10-2025.png|527]]
	- Если мы говорим об отстающей цепи, то там этот комплекс формируется в начале синтеза каждого фрагмента Оказаки


## Виды ДНК-полимераз

- Эукариотические ДНК-полимеразы
	- Альфа
		- В ней есть субьединица праймазы - синтезирует праймер
		- Другие субьединицы работают как полимераза, добавляют около 20 нуклеотидов, после чего диссоциирует
	- Бетта
		- Для восстановления ДНК
	- Дельта
		- Для синтеза отстающей цепи
		- Есть 3’-5’ экзонуклеазная активность
	- Эпсилон ε
		- По большей части для синтеза лидирующей цепи
		- Она прикрепляется с одной стороны к PCNA, с другой - к хеликазе
			- Благодаря этому может долго ехать по ДНК, что и нужно для синтеза лидирующей цепи
- Прокариотические ДНК-полимеразы
	- ДНК-полимераза 1
		- Для восстановления ДНК, также вырезает фрагменты Оказаки
		- Из всех полимераз чаще всего работает эта
	- ДНК-полимераза 2
		- Для репарации
	- ДНК-полимераза 3
		- Основная, как дельта у эукариот
	- и еще есть 4 и 5, странные ребята

- ДНК-полимеразы прокариот используют в [[ПЦР]]
