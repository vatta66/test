# Решение задачи через аналогичную

- Method for [[Решение проблемы]]
- Придумайте аналогию к задаче
	- To do it you can use [[Анализ]]
	- Also you can use [[Моделирование]]
- Далее развивайте ее, смотрите, что бы вы могли сделать с объектом аналогии
- Дальше посмотрите, можете ли вы совершить такое же действие с задачей
- Перебирайте варианты
- You can use [[метафоры]]
