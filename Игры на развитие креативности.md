# Игры на развитие креативности

 - Ассоциации. Сидят четыре человека, они разбиты по парам. Человек из одной пары загадывает слово человеку из второй пары, тот одним словом должен объяснить его члену своей команды. Если он не угадывает, человек, который загадал слово объясняет его члену своей команды. Можно использовать слова, в которых будут зашифрованы правильные ответы
 - Say the same. Два человека одновременно говорят слово, дальше они должны с помощью ассоциаций сказать одинаковое слово
 - Связки существительное-прилагательное так, чтобы они были невообразимы вместе
