# Аналитическая химия
Up:
Next:

- Методы определение хим состава в-ва и его структуры
- Разделы
	- [[Качественный анализ]]
	- [[Количественный анализ]]
		- Соотношение частей вещества
- Классификация анализов 2
	- Элементный
		- Определяем конкретные элементы, но нам все равно в каких они соединениях
	- Вещественный
		- Выясняем конкретные вещества, а их элементы не важны
- Методы аналитической химии
	- В зависимости от массы вещества
		- Макрометоды - нужно не менее 0.1г в-ва
		- Полумикрометоды - нужно 0.1-0.01г в-ва
		- Микрометоды - от одной сотой до одной тысячной
		- Ультрамикрометоды - около десяти в минус шестой
		- Субмикрометоды - около десяти в минус девятой
	- В зависимости от вида анализа
		- Методы разделения
			- Делим смесь на составные части и определяем их
			- Пример: хроматография
		- Методы определения
			- Делим состав самой смеси
	- химические, физиеские и физико-химические методы
		- Реакция ионного обмена с осадком - химический метод
			- всегда в основе хим реакция
		- Масспектрометрия, УФ - физический
		- Кондуктометрия - физико-химический
			- Там суть в том, что через вещество проводят ток
	- В зависимости от агрегатного состояния
		- Сухой метод
			- Окрашивание пламени (чаще всего для катионов)
			- Метод образования окрашенных перлов
				- При сплавлении с тетраборатом натрия или фосфатной солью некоторые вещества дают окрашенные стелка - перлы
			- Метод растирания порошков
				- При смешивании двух порошков происходит окрашивание
		- Мокрый метод
			- С растворами
- Правила выбора метода анализа
	- Чувствительность - сколько вещества надо
	- Избирательность
	- Точность
	- Экспрессность (быстрота)
- Аналитические сигналы
	- Цвет, запах, выделение газа, окрашивание пламени, образование люминесцирующих соединений, выпадение или растворение осадка
	- Важна интенсивность сигнала
		- Она связана с концентрацией вещества: P = f(c), p - интенсивность
		- Благодаря этому можно по интенсивности с помощью градуировочного графика определить его концентрацию
