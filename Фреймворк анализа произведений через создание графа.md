# Фреймворк анализа произведений через создание графа

- Создается заметка произведения
	- В ней указывается ссылка на автора, основные и прочие идеи
- Создаются заметки-MOCи Основные идеи произведения и Прочие идеи произведения(остальные идеи и мысли, которые затронул автор в своем произведении)
- По ходу прочтения/просмотра выписываются все ключевые детали сюжета и персонажи
	- Детали сюжета
		- Деталями могут быть как события сюжета, так и средства, которыми они создаются(например, цвета)
		- Если это событие сюжета, заметка связывается с персонажами
			- Можно указать причины возникновения события(допустим, ссылка на характер персонажа)
		- Деталь связывается либо с другой деталью, либо с основные/прочие идеи
	- В замекте каждого персонажа указывается 
		- Особенности описания (внешность, его сравнения с кем-либо/чем-либо)
			- По возможности каждая особенность внешности связывается с особенностью характера
			- Особенности характера связываются с деталями произведения(допустим, алчность персонажа связывается с событием, где она проявилась)
		- Его роль относительно основные/прочие идеи (связывается не с МОС-ом, а с заметкой конкретной идеи)
			- Т.е. для раскрытия каких идей автор использует этого персонажа
			- Указывается с помощью 
				- Связи деталей сюжета, в которых принимал участие персонаж, и основными/прочими идеями
				- Насколько успешен персонаж в произведении, какое место занимает по его окончанию
		- Можно связать персонажа с MOC-ом его архитипа, если таковой имеется
			- Такми образом устанавливаются связи между произведениями
	- Заметка каждой идеи может быть связана с замеками похожими идеями из других произведений/заметками концепций, на которые она опирается(философские, психологические и тд)
- Противоположные идеи:
	- Линейная система анализа в школах
- Из чего это следует:
	- Необходимость такого подхода обуславливается
		- Низким уровнем заинтересованности школьников в чтении
		- Низкой способностью школьников к анализу
	- [[Мой проект по внедрению системы цеттелькастен в школы]]
	- Эффективность такого подхода обуславливается [[Искусственная актуализация знания]]
- Что из этого следует:
	- Более высокая эффективность запоминания и понимания произведения
