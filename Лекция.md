# Лекция

- Один из [[методы обучения]]
- Преподаватель сообщает материал, а ученики его активно потребляют
- Сущетсвуют разные формы лекции
	- Вводная лекция
		- Общие сведения по предмету, его роль в мире, истории, связь с другими дисциплинами
	- Проблемная лекция
		- Изложение материала с помощью проблемных вопросов
	- Бинарная лекция(лекция-диалог)
		- Изложение материала происходит в виде диалога двух специалистов(например, теоретика и практика, представителей двух научных направлений и тд)
	- [[Лекция-провокация]]
	- Лекции-конференции проводятся в виде слушания докладов
	- Лекции-консультации предполагают изложение материала по принципу вопросы-ответ или вопрос-ответ-дискуссия
