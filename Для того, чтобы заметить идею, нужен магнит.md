# Для того, чтобы заметить идею, нужен магнит

- Для того, чтобы заметить идею в массе [[информация]], нужно обладать магнитом, который обеспечит притяжение этой идеи
	- Это может быть конкретная цель: найти информацию по теме х
	- Это может быть система ваших знаний - [[Знания в системе образуют магнит, притягивающий другие знания]]
		- Если у вас нет определенного знания, вы можете не заметить какую-то важную информацию в массе
	- Это может быть [[Состояние идейной осознанности]] 
	- Это может быть ваш личный опыт
