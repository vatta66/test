# Классификационные заметки - только средство для связи двух идей
Up: [[цеттелькастен]]
Next: [[Классификационное знание]]

- Если у вас есть заметка с классификацией, то есть просто перечисление
	- Как, например, заметка [[факторы производства]]
- У вас скорее всего не получится сделать ссылку из нее на какую то другую заметку, используя граф анализ или беклинки
- Потому что из нее нет никаких следствий, только например то, что капитал является фактором производства
- Но такие заметки можно использовать для связи двух других заметок из них самих
	- То есть вы находитесь в какой то заметке, которая ссылается на классификацию, и с помощью граф анализа связываете ее с заметкой, которая также ссылается на нее
	- Заметка-классификация выступает в роли посредника
