# Жизненный цикл [[покрытосеменные]]
[[Жизненные циклы растений]]
- Спорогенез (похож на [[голосеменные]])
	- Микроспорогенез происходит в пыльнике тычинки ([[Цветок#^6d4819]])
	- Макроспорогенез происходит в семязачатке пестика ([[Цветок#^8945c9]])
		- В ходе макроспорогенеза образуется одна макроспора (в ходе [[мейоз]]а материнской клетки мегаспоры образуется еще 3 редукционных тельца
- Гаметогенез
	- В ходе микрогаметогенеза образуется пылинка (гаметофит), состоящая из вегетативной клетки и двух спермиев (3 клетки всего)
	- В ходе макрогаметогенеза макроспора делится [[митоз]]ом 3 раза - образуется 8 клеток (nc)
		 - Три клетки у входа в зародышевый мешок (вход - микропиле) - 2 синергиды и одна яйцеклетка
		- Две клетки посередине сливаются в одну диплоидную - центральная клетка
		- Три клетки в конце - антиподы
		- Гаметофит - восьмиядерный зародышевый мешок
- Опыление
	- В зависимисти от [[виды опыления]], пылинка либо с помощью насекомого, либо с помощью ветра садится на рыльце пестика
		- Затем, ее вегетативная часть прорастает внутрь столбика, образуя пыльцевую трубку
			- Два спермия опускаются по ней
	- Происходит двойное оплодотворение
		- Один спермий сливается с яйцеклеткой, образуя диплоидную зиготу - будущий зародыш
		- А другой - с центральной клеткой, образуя триплоидный эндосперм
		- Увеличение уровня плоидности эндосперма до 3n способствует более быстрому росту ткани по сравнению с диплоидными тканями спорофита
- Затем, семя прорастает - [[Base/Ботаника#Семя]]
