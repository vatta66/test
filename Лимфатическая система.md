# Лимфатическая система

[[Межклеточная жидкость+кровь+лимфа.excalidraw]]

- Из артериального [[Капилляры]] под большим давлением часть плазмы крови выходит из него в межклеточное пространство
	- Откуда попадают в лимфатический сосуд
	- Лимфа - жидкая часть плазмы крови
- В [[Органоиды клетки|клетке]] накапливаются продукты обмена, которые выходят из нее наружу
	- Попадают в межклеточное пространство
		- Плазма крови, просочившаяся сквозь артериальные капилляры, захватывает метаболиты и всасывается в лимфатический сосуд
- В лимфе содержатся [[Лимфоциты]]
- Лимфатические капилляры
	- Начинаются слепо
	- Нет четкого направления тока лимфы
	- Нет четкого диаметра, расширенные и суженные участки, выпячивания, отростки
- Лимфатические капилляры переходят в лим. сосуды
	- Имеют клапаны для направления тока
		- ![[Pasted image 20230505183317.png|339]]
- Сосуд идет в [[лимфатический узел]]
- Далее, лимфатические сосуды впадают в подключичные вены
