# Головоногие [[моллюски]]
Класс Cephalopoda < тип [[Моллюски]]

- [[Строение головоногих моллюсков]]
- Подклассы
	- Вымершие подклассы
		- подкласс Bactrites
			- Была раковина в виде башни
				- Из за нее не могли быстро передвигаться, поэтому и вымерли
		- подкласс Ammonoidea
			- Раковина бактритесов закручивалась, и получились аммониты - раковина у них закручена в спираль
		- подкласс Belemnoidea
			- У них была внутренняя раковина - т.е. снаружи покрыта мантией
			- Фрагмокон - часть раковины, где находилось тело молллюска
				- Разделен на камеры
			- Рострум - массивная часть раковины 
			- Проостракум - к этой части раковины прикреплялась мускулатура
	- подкласс Nautiloidea (четырехжаберные)
		- Представитель – [[Наутилус]]
	- подкласс Coleoidea (Двужаберные)
		- Не имеют внешней раковины (либо имеют сильно редуцированную внутреннюю)
		- Надотряды
			- Octobrachia (восьмирукие)
				- 8 щупалец
				- Раковины полностью нет
				- Отряды
					- о. Octopoda (осьминоги)
						- Синекольчатый осьсминог
							- ![[Pasted image 20240223163032.png|180]]
							- Используют сильнейший нейротоксин (яд)
								- Одно из самых ядовитых животных в мире
							- ![[Pasted image 20240223163230.png|170]]
						- ![[Головоногие моллюски-01-08-2025.png|579]] осьминог с потомством
							- сильно заботятся о нем
					- о. Argonauta (аргонафты)
						- Самки до 12см
							- Имеет временную раковину для вынашивани потомства
						- Самцы 2-3см
					- о. Vampiromorpha (вампироморфы)
						- Есть перепонки между щупальцами
						- Пока только один вид
			- Decabranchia (десятирукие)
				- 10 щупалец
				- Отряды
					- о. Sepiida (каракатицы)
						- Придонные хищники
						- Тоже могут изменять окраску
						- Есть внутренняя раковина
							- Сплющенна
					- о. Teuthida (кальмары)
						- Раковина внутренняя, представляет из себя стержень - гладиус
							- По сути не нужна
						- Architeuthis
							- Самый большой вид кальмаров
							- Максимальный размер не зафиксирован (15+ метров)
							- Оставляют на кашалотах шрамы от борьбы
