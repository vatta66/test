# Внутриколлективный отбор

- Если создать довольно изолированную систему со своими коллективами внутри, то возможно ли будет создать явления, схожие с направлением эволюции [[Продукты внутривидового отбора]](только вперед)?
	- Допустим, направить отбор на принцип чистоплотности в коллективе([[Принцип связи воспитания с жизнью]])
	-  Моия идея позволят добиться этого - [[Мой проект Создание школ-интернатов как основа воспитания]]
- Как это будет работать
	- Педагог начинает различными способами культивировать черту(например, [[Использование эффекта сессии побед и соревнования между учениками для создания положительных привычек]]), самое главное - создать установку внутри коллектива о том, что что-то плохо, а что-то - хорошо(в основном это будет происходить с помощью [[Положительное подкрепление]])
		- Далее, в зависимости от уровня сформированности коллектива([[Этапы формирование детского коллектива]]), с разной силой будет работать [[Принцип параллельного действия в коллективе]]
- До какого-то возраста человек должен находиться в изолированной системе для принятия истинных(нужных) убеждений(нравственных, просоциальных и тд)
	- Например, идет взаимодействие между такими же изолированными коллективами, в которых культивируются те же убеждения
		- Такое взаимодействие будет являться [[Воспитывающая ситуация]] для этих коллективов
- Далее начинается более активное взаимодействие с внешним миром и тестирование убеждений
	-  Нужно обеспечить их учтойчивость, чтобы те не посыпались при встрече с реальностью
		- Например, с помощью [[Эффект вакцинации]]
- Если все поколение воспитывать по этому принципу, любое взаимодействие человека будет проходить с такими же нравственными, просоц. людьми
- Из чего это следует:
	- Знания о внутривидовом отборе + невозможность зарождения альтернативных идей в хорошо контролируемом изолированном коллективе
- Что из этого следует:
	- Мы можем намного эффективней 
		- Прививать любые положительные [[Привычка]] и [[Потребность]] детям в условиях таких коллективов
			- Тем самым осуществляя [[воспитание]]
