# Бороздчатобрюхие
Up:
Next:
класс Solenogastres < Apcacophora < подтип Amphineura < тип [[Моллюски]]

- Морские, червеобразное тело
- Бентосные, но не зарываются
	- Обгладывают колониальных [[Гидроидные]]
- Тоже есть кутикулярные шипы, но выражены слабее
- Нога есть, но слабо выражена
	- В виде борозды
	- Она выстлана реснитчатым эпителием, с помощью которого они и ползают
- Строение
	- В области головы ресничная ямка - хеморецепторный орган 
	- Короткая средняя кишка, задняя длиннее
	- У них нет ктенидиев - дышат всей поверхностью тела
