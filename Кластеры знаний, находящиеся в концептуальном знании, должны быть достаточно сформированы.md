# Кластеры знаний, находящиеся в концептуальном знании, должны быть достаточно сформированы

- [[Кластер знаний|Кластеры знаний]], на основе которых построены [[Понятие|понятия]], входящие в [[Классификационное знание]], должны обладать достаточным [[Коэффициент кластеризации]] и быть в достаточной степени созревшими ([[Процессинг кластера знаний]])
	- Таким образом, у ученика появится ощущение о[[смысл]]енности данного понятия для него лично и, возможно, [[Познавательный интерес]]
- В противном случае, концептуальное знание не будет иметь [[смысл]]а для ученика
- При соблюдении этого принципа, концептуальные знания могут стать эффективным способом включения кластеров в [[Сеть знаний]]
- В настоящее время мы наблюдаем нулевую сформированность кластеров знаний понятия, т.е. ученикам просто дают его определение и парочку фактов (и то не всегда)
	- Например, если кластер знаний о репликации днк сформирован недостаточно, концептуальное знание о 4 принципах репликации не будет иметь для ученика смысла. В правильном варианте, каждый принцип должен представлять из себя небольшой кластер, плотно включенный в большой кластер знаний о репликации
	- Но, если у ученика в голове нет ничего, кроме факта “у репликации есть 4 принципа: …”, он не догадается [[декомпозиция|декомпозировать]] это знание, потому что, согласно [[Принцип достаточной атомарности]], для него это не будет иметь смысла
		- Объяснение: из-за того, что классификация не является кластером, а воспринимается учеником как факт “у репликации есть 4 принципа:…”, она является атомарным знанием для ученика
			- Из за этого он не может подумать о том, чтобы декомпозировать его дальше
