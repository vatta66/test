# Объяснение потребности в самодетерминации через теорию когнитивного диссонанса

- Возможно, [[Потребность в самодетерминации]] можно объяснить с помощью [[Когнитивный диссонанс]]
- Для объяснения поведения(поход на занятия) в одном случае используется только интерес, что соответственно ведет к еще большей любви к занятиям, а во втором - еще и [[Base/Деньги]], которые постепенно вытесняют причину в виде интереса
	- Тажке во втором случае интерес будет падать и из-за неинтересности самих тем, тогда как в первом этот эффект будет слабее из-за самоубеждения с целью разрешить когнитивный диссонанс
