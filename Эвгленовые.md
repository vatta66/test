# Euglenozoa
Эвгленозои

- Характеристика
	- Два жгутика выходят из резервуара
		- Один длинный и толстый
			- Толстый за счет параксиального тяжа
				- ![[Эвгленовые-30-09-2024.png|365]]
			- Для перемещения
			- У его основания есть флагеллярное вздутие - в нем расположен флавиновый фоторецептор
		- Второй короткий, не выходит за пределы резервуара
			- Для подгребания пищи к цитостому
		- Но бывают эвглены с 4 и даже 7 жгутиками
	- Оболочки - [[Типы оболочек протист|кутикула]] или [[Типы оболочек протист|тубулемма]]
	- Цитостом (клеточный рот)
		- Находится на дне резервуара
	- Сократительная вакуоль
		- Служит для выбрасывания продуктов метаболизма и излишка воды
		- На дне резервуара
	- У трети видов есть трехмембранный [[Хлоропласты]]
	- Дисковидные кристы [[митохондрия]]
	- Запасное вещество - парамилон
- Систематика
	- н/кл Euglenida
		- [[Эвглена зеленая]]
	- н/кл [[Kinetoplastida]]
