# Перья [[Птицы]]

- Являются видоизмененными чешуями [[Пресмыкающиеся]]
	- Поэтому состоят из керотина
- Контурные перья
	- Строение контурного пера
		- Нижняя часть - очин, пушистые отростки - опархало
			- Опахало состоит из бородок 1-ого порядка, которые крепятся напрямую к стержню, и бородок 2-ого порядка, расположенных на бородках 1-ого порядка
				- У бородок 2-го порядка есть крючочки, которыми они сцепляются с бородочками соседней бородки
					- Такое сцепление делает опахало монолитным и гладким
		- ![[Pasted image 20230711183713.png|300]]
	- Виды контурных перьев
		- Маховые
			- Растут по краю крыла
			- Поддерживают птицу во время полета
		- Рулевые
			- Являются продолжением хвоста
			- Помогают в изменении направления полета
		- Покровные
			- Формируют обтекаемый контур птицы
- Пуховые перья
	- На бородках нет крючков
		- Опахало не плотно сцепленное
	- Функция - теплоизоляция
	- ![[Pasted image 20230711184951.png|100]]
- Функции:
	- Перья обеспечивают обтекаемость тела, уменьшая сопротивление воздуха при полёте
	- Перья способствуют теплоизоляции, а у некоторых водоплавающих видов и гидроизоляцию
		- Для обеспечения гидроизоляции перья смазываются гидрофобным секретом копчиковой железы
	- Перья, расположенные на крыльях и хвосте, в полёте образуют несущую плоскость
	- Перья используются самцами для привлечения самки ([[Принцип гандикапа]])
	- Пуховые перья удерживают тепло при насиживании кладки яиц
