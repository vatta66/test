# Способ обучения мыслительной операции

- Для того, чтобы сформировать [[навык]] [[Мыслительные операции]] у ученика, при объяснении материала нужно 
	- Активно использовать данную мыслительную операцию на уроках
		- Одновременно лучше развивать только одну - [[Формируйте один аспект поведения за раз]]
	- Во время объяснения обращать [[Внимание]] аудитории на применяемый вами метод
		- Например, указывать ученикам, какие именно ссылки вы делаете, какую модель сейчас используете и тд
		- Будет происходить [[Повышение важности идеи при направлении на нее внимания]]
- Также нужно осуществить [[Привычка обращать внимание на определенные вещи]] - для каджого случая они будут свои
- В обучении ученика мыслительным операциям помогут
	- [[Проблемное обучение]]
	- [[Исследовательский метод в педагогике]]
- Таким же образом можно развивать [[Навык системного мышления]]
