# Давайте ученикам решенные задачи с ошибками
Up: [[Домашнее задание]]
Next: [[Кластер ошибок учеников]]

- Вы можете дать ученикам решенные задачи, в которых уже есть ошибки. Дети же должны найти их и написать правильный ответ
	- Чем чаще перед ребенком будет стоять задача в поиске ошибок, тем быстрее он научится находить их у себя
- Будет вырабатываться [[Привычка обращать внимание на определенные вещи]] - на ошибки
- Выгодно будет провести такое задание перед проверочной, чтобы сделать [[прайминг]]
- Таким образом, ученик с большей вероятностью найдет ошибку в своем решении
- Для русского языка, математики, химии, физики и др
- Можно использовать в формате проверочной “найди все ошибки” - [[тестирование в педагогике]]
