# Трентеполиевые
Up: [[Зеленые водоросли]]
Next:

- Наземная водоросль
	- Приспособления для этого
		- Мощная [[клеточная стенка]] из множества слоев для предотвращения высыхания
		- Большое количество каротиноидов: B-каротин и [[Астаксантин]], которые маскируют хлорофилл
		- В хлоропластах отсутствуют пиреноиды
- Может быть эпифитом или литофитом
	- Либо на коре [[Покрытосеменные|деревьев]], либо на влажных камнях
- Экология
	- Может быть [[симбиоз|симбионтом]] [[лишайник]]а
	- род Cephaleuros вызывает ржавчину растений
