# Антидиуретический гормон (вазопрессин)
tags: #bio

- Синтезируется в [[Гипоталамус]] в ответ на повышенную концентрацию солей в организме 
- Эффекты
	- Увеличение проницаемость [[Мочевыделительная система человека|дистального канальца]] и [[Мочевыделительная система человека|собирающей трубки]] для воды ([[Мочевыделительная система человека|реабсорбция]] воды)
	- Через кору запускает чувство [[голод и жажда|жажды]]
	- При большой концентрации способен вызывать [[Вазоконстрикция]] (за что и был назван вазопрессин)
- [[Алкоголь]] ингибирует выделение АДГ(одна из причин головной боли). 
- Принцип работы
	- Работает через [[Аденилатциклазная система]]
	- В клетке соб. трубк. есть везикулы с встроенными в них белками - **аквапоринами**
	- При действии АДГ эти везикулы встраиваются в мембрану, увеличивая тем самым проницаемость для воды
	- Также есть вариант длительной регуляции, при котором запускается экспрессия [[ген]]ов аквапорина
