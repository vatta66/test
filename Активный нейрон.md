# Активный нейрон

- Нейрон с точки зрения [[Теория функциональных систем]]
- Процесс работы
	- Приход возбуждения 
	- Выход [[Нейромедиатор]] 
	- Действие нейромедиатора на [[синапс|постсинаптическую мембрану]] 
	- Метаболические процессы в области постсинаптической мембраны
	- Интегративные метаболические процессы в соме нейрона
	- Формирование [[Потенциал действия]]
- Все межклеточные контакты служат обмену метаболическими субстратами между контактирующими клеточными образованиями
	- [[Нейромедиатор]]ы служат для метаболизма [[нейрон]]а, а не для передачи [[Потенциал действия]] (т.е. он не основная цель)
		- Натрий, при входе в клетку, переносит за собой [[Глюкоза]] ПРОВЕРИТЬ
		- Также, нейромедиаторы являются активаторами [[Протеинкиназа А]] через [[Аденилатциклазная система]] (в случае [[Метаботропный рецептор]]), которая нужна для синтеза белков в клетке
		- В метаботропных нейронах нейромедиаторы напрямую изменяют метаболизм нейронов через
- Как и любая другая клетка, [[Нейрон]] для реализации своей генетической программы нуждается в метаболитах, поступающих от других клеток (в том числе от [[Глиальные клетки]], из крови)
