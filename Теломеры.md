# Теломеры

- У людей последовательность TTAGGG
	- Повторяется примерно 1000 раз
- Они укорачиваются с каждой [[Репликация ДНК]], выполняя роль своеобразной заглушки. 
- За создание теломер отвечает фермент теломераза
	- Теломераза состоит из нескольких доменов белка и из РНК
	- РНК является матрицей для синтеза ДНК
		- Получается, что теломераза работает как обратная транскриптаза
	- ![[Теломеры-23-10-2025 1.png|300]]
- ДНК способно реплицироваться без потери генетической информации примерно 50 раз - это **Предел Хейфлика**.
- С укорочением теломер связана одна из теорий старения - [[Теория укорочения теломер]]

- Проблема сшивания концов хромосом
	- Ферменты репарации могут воспринимать концы хромосом как поврежденные участки ДНК, и сшивать гомологичные и негомологичные хромосомы вместе
	- Чтобы этого не происходило, теломеры образуют петли
		- Теломераза достраивает только 5’-3’ конец ДНК, то есть он получается длиннее чем второй
		- Этот длинный конец загибается и связывается с белками - шелтеринами
		- ![[Теломеры-23-10-2025.png|619]]
