# Общество
Up:
Next:

- **Общество** - обособившаяся от при­роды, но тесно с ней связанная часть материального мира, представ­ляющая собой формы объединения и способы взаимодействия людей
- В узком смысле, **общество** - определенный этап исторического развития
- Признаки общества как системы
	- **Открытость**
		- Взаимодействует с внешним миром, берёт оттуда нормы, правила, порядок действий, ценности и прочее
	- **Иерархичность**
		- Основные подсистемы общества → основные комплексы институтов → [[Социальный институт]] (см. [[структура общества]])
		- [[Иерархичная структура общества]]
	- **Функциональность**
		- Общество и его институты выполняют ряд задач для удовлетворения общественных потребностей
	- **Динамизм** ^c6cabc
		- Общество и его элементы изменяются со временем (мода, традиции, нормы)
	- **Целостность**
		- Работая вместе люди достигают бОльших результатов
	- **Устойчивость**
		- Общество обладает механизмами поддержания ранее сложившихся структур, если кто-то выбивается из общества (ведёт себя необычно, привлекает внимание), то общество накладывает [[санкции]]
	- **Интегративность**
		- Компоненты общества связаны между собой (в т.ч. с помощью интернета)
- Признаки общества как динамичной системы
	- Незавершенность развития
		- Общество не перестанет развиваться
	- Нелинейность развития
		- Общество развивается скачками
		- С помощью [[революция]] (неолетическая, промышленная, научно-техническая революции)
	- Альтернативность развития
		-  У общества есть разные пути развития, которые выбираются в зависимости от
			- Природных факторов
			- Социальных факторов
			- Уровня развития науки и техники

	- Воспроизводство и [[социализация]]
