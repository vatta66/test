# Стебель
Up: [[Органы растений]]
[[Побег.excalidraw]]

- Один из [[Органы растений]]
- [[Клеточное строение стебля]]
- [[Видоизменения побега]]
- Газообмен происходит через [[Устьица|устьица]] и [[чечевички]]
### Функции стебля
1. Проводящая
2. Опорная
3. Несет [[лист]], [[Почки растений]], [[Цветок|цветок]]

### Строение стебля
1. **Узел** - место, от которого отходит [[лист]]
2. **Междоузлие** - участок между двумя узлами
3. **Пазуха** - угол между листом и стеблем

- Стебель бывает **травянистый** и **одревесневший**
	- **Молодой травянистый** стебель: эластичный, зеленоватого оттенка, покрыт эпидермой, способен к [[Фотосинтез]]у
	- **Одревесневший зрелый** стебель подвергается одревеснению, покрыт перидермой. Содержит большое количество древесины
