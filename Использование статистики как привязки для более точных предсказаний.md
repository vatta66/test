# Использование статистики как привязки для более точных предсказаний

- Solution of problems [[Пренебрежение априорной вероятностью]] and [[Ошибка планирования]]
- If you want to make good prediction, you should use statistic as an adjustment point ([[Эффект привязки]]), and then increase or decrease it relying on your specific data
	- “базовый прогноз должен быть привязкой — отправной точкой для будущих корректировок.” ([Канеман, p. 253](zotero://select/library/items/6V3HYD2Z)) ([pdf](zotero://open-pdf/library/items/LU9XSCWF?page=253&annotation=8WF6LR5V))
		- 1. Определите соответствующую исходную категорию (перепланировка кухни, крупный железнодорожный проект и тому подобное). 
		- 2. Добудьте статистические данные по исходной категории (затраты в пересчете на милю железнодорожных путей или долю перерасхода бюджета в процентах). Используйте статистику для построения базового прогноза. 
		- 3. Используйте конкретную информацию для модификации базового прогноза
	- Also, we can take prior probability as a adjustment point for [[оценка вероятности]]
	- Useful for [[Принятие решения]]
- Also, it should be used in different areas because of [[Опасность индивидуального подхода]]
- [[Эффективное мышление MOC]]
