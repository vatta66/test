# Класс Двудольные из отдела [[Покрытосеменные]]

- 2 семядоли
- Количество лепестков в [[цветок]] кратно 4 или 5
- Запас питательных веществ - в семядолях
	- Исключения: томат, мак, тыква, подсолнечник
- Есть [[Клеточное строение стебля#^de4268|камбий]]
	- Следовательно, происходит вторичное утолщение [[стебель]]
-  На срезе корня - _звезда_ ![[Pasted image 20220324135351.png|296]]
- На срезе стебля - кольцо из _проводящих_ пучков ![[Pasted image 20230615144856.png]]
- [[Стержневая корневая система]]
- [[Цветок#Виды околоцветников]] Двойной околоцветник
- Жилкование пальчатое или перистое ![[Pasted image 20230628160552.png]]
- Представлены все жизненные формы
- Семейства
	- Крестоцветные
		- Капуста, редька, редис, пастушья сумка
		- Плод - стручок
		- Соцветие - кисть ![[Pasted image 20230615170811.png]]
		- Содержат много [[Лизоцим]]а
		- Нижние листья могут образовывать розетку
		- [[Цветок]] четырёхчленного типа
	- Розоцветные
		- Шиповник, яблоня, манжетка, вишня, розы
			- В семечках есть цианид. Зачем?
		- Плод - костянка (![[Pasted image 20230702164932.png]]), яблоко, ягода
		- Соцветие - щиток, кисть, простой цветок, простой зонтик
		- ![[Pasted image 20230615174807.png]] ![[Pasted image 20230615174839.png]]![[Pasted image 20230615170811.png]]
		- Пятичленный тип цветка, двойной околоцветник
	- Пасленовые
		- Картофель, дурман, белена, табак, томат
		- Плод - ягода или коробочка (![[Pasted image 20230702164507.png]])
			- Плод картофеля - ягода, а [[клубень]] картофеля - [[Видоизменения побега]]
		- Соцветие - кисть ![[Pasted image 20230615170811.png]]
		- Отличительная особенность палсленовых - выделение [[никотин]]а как защиты от колорадского жука
	- Бобовые
		- Фасоль, горох, соя, [[Клевер ползучий.excalidraw|клевер]], бобы, [[акация]], вика
		- Плод - боб
		- Соцветие - головка или кисть  ![[Pasted image 20230615175326.png]] ![[Pasted image 20230615170811.png]]
		- Для них характерны [[Видоизменения корня#^5bd866|клубеньки]]
	- Сложноцветные (астровые)
		- Одуванчик, василек, георгин, подсолнечник
		- Плод - семянка ![[Pasted image 20230702164740.png]]
		- Соцветие - корзинка ![[Pasted image 20230615175606.png]]
		- ![[Двудольные растения-15-08-2024.png|289]]
			- Каждая маленькая точечка в центре - это цветок - [[Цветы сложноцветных.excalidraw]]
	- Тыквенные
		- Арбуз, дыня, тыква, кабачки, огурец, бешеный огурец
		- Плод - тыквина
	- Кувшинковые
		- [[Кубышка желтая.excalidraw]]
