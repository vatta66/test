# Ресничные черви
Класс Turbellaria в типе [[Плоские черви]]

- Свободноживущие хищники
- Тело листовидное уплощенное
- Большинство морские или пресноводные
	- Есть и сухопутные, в тропических лесах
		- Достигают 40 см (максимум для турбеллярий)
- Кожа густо покрыта ресничками
	- Служат для того, чтобы перегонять воду вокруг тела
		- Дыхания как такового нет, поэтому, для того, чтобы газообмен шел, они вынуждены постоянно прогонять воду вокруг своего тела
- Много желез
	- Рабдитные железы
		- Внутри них формируются рабдиты
			- Пластинки, которые при контакте с водой дают огромный объем слизи
			- Защитная функция
				- Выбрасывают их в нападающего
			- Облегчает передвижение
	- Дуогландулярные железы
		- Два типа клеток
			- Выделяющие клейкие вещества, прикрепляющие червя к субстрату
			- Выделяющие растворитель
- ![[Покровы турбеллярий]]
	- Клетки паренхимы
		- Мезодермального происхождения
		- Почти все являются [[стволовые клетки]]
- Мускулатура [[Мускулатура червеобразных|кольцевая, продольная, поперечная]]
- Мультицилиарный эпителий + мускулатура = кожно-мускульный мешок
- [[Нервная система лестничного типа|Нервная система - ортогон]] (ортогональная, лестничная)
- Пищеварительная система
	- Замкнутого типа
	- Состоит из глотки и средней кишки
	- Ротовое отверстие с нижней стороны тела
	- Т.к. кровеносной системы нет, пищеварительная сеть должна быть очень большой
		- Состоит из нескольких желудочных каналов
			- С выростами - дивертикулами
				- Пронизывают все тело насквозь
- Протонефридиальная система
	- Протоки [[Протонефридии]] соединяются в единые мочевыводящие каналы
		- Которые октырваются наружу нефропорами (экскреторные поры)
- Продукты азотного обмена - NH3
	- Выделяется не через протонефридии, а через кожу
- Регенерация
	- Из отрезанных частей вырастают новые черви
	- Если сделать надрез не до конца, может вырасти еще одна голова
		- В зависимости от количества надрезов
- Размножение
	- Благодаря такой способности к регенерации активно пользуются [[Виды размножения|бесполое размножения]]
		- Паратомия
			- Похож на долгое почкование
			- На заднем конце образуется цепочка из почек
				- Каждая постепенно развивается в червя
				- Отпочковывается, когда полностью сформируется
			- Одна из гипотез происхождения [[Ленточные черви]] - незавершенная паратомия
		- Архитомия
			- Тело червя разделяется на фрагменты, после чего каждый из достраивается до полноценного червя
			- Похож на фрагментацию
		- Деление
			- Червь делится на две части, которые развиваются в новых червей
	- [[Виды размножения|Половое размножение]]
		- Являются гермафродитами
			- Половые органы - гонады, находятся в паренхиме
		- Внутреннее оплодотворение
			- Тот червь, который в ходе [[Оплодотворение]] выступает в роли самца, прокалывает пенисом стенку другого червя и вводит сперму ему в первичную полость
				- Пенис - самая жесткая часть организма
					- Используются также при охоте
						- Может заколоть жертву пенисом
		- При спаривании, каждый червь хочет быть самцом
			- Для решения этого вопроса есть ритуал - фехтование пенисами (penis fencing)
				- При котором каждая особь пытается осеменить другую
		- Зачастую, самка умирает
			- Вылупившиеся червячки доедают ее остатки
- Реакция на неблагоприятные условия
	- При недостатке кислорода или в случае сильного повышения температуры воды планарии сами распадаются на куски, которые регенерируют при наступлении благоприятных условий
- Систематика
	- отр. Acoelomorpha (бескишечные)
		- Глотка на переднем конце тела
		- Нет протонефридий, кишечника
		- Есть временная кишка
			- Скопление энтодермальных клеток, формирующих временную пищеварительную полость
		- Нервная система диффузного типа
		- ![[Ресничные черви-11-11-2024.png|237]]
		- Представитель - Symsagittifera roscoffensis
			- Вступает в симбиоз с [[зеленые водоросли]] _Tetraselmis convolutae_: на стадии личинки поедает их, и затем живет всю жизнь за счет их фотосинтеза
			- Но умирает, т.к. водоросли тоже начинают умирать
	- отр. “Rhabdocoela” (прямокишечные, рабдоцерии)
		- Крупнее ацелеоморф, до пары сантиметров
		- После глотки есть средняя кишка
		- Представители
			- род Микростомум
				- Может воровать [[Стрекательная клетка|стрекательные клетки]] у [[гидроидные]] - клептокниды
			- группа temnocephalida
				- Облепляют [[ракообразные]], имеют нетипичные щупальцевидные выросты
				- ![[Ресничные черви-11-11-2024 1.png|300]]
	- отр. Tricladida (трехветвистые)
		- Активные хищники
		- Появляются глаза
		- Глотка располагается посередине брюшной стороны тела
			- Т.к. они крупные, они наползают на жертву и поедают ее
		- От глотки отходит трехветвистая средняя кишка
			- Одна ветвь идет от глотки к передней стороне тела, две - к задней по разным сторонам тела
			- ![[Ресничные черви-11-11-2024 2.png|300]]
		- Представители
			- Молочнобелая планария
				- ![[Pasted image 20230103171554.png|568]]
			- Microplana termitophaga
				- Охотятся на [[термиты]]
					- С помощью клейкого секрета на переднем конце тела хватают их, потом перемещают к глотке и переваривают
				- [Zimworms - YouTube](https://www.youtube.com/watch?v=ZpXSz8byR_s)
	- отр. Polycladida (многоветвистые)
		- Глотка по середине тела
		- От нее - многоветвистая средняя кишка
		- ![[Pasted image 20230103171626.png|534]]
