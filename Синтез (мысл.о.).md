# Синтез

- Соединение элементов изучаемого объекта в единое целое
- На основе наиболее важных [[Атомарное знание]], сравненных друг с другом (после п. [[сравнение]]), а также с другими знаниями, учащиеся формулируют новые [[Атомарное знание]]
- На основе этих новых атомарных знаний и исходных знаний темы может осуществляться [[Обобщение]]
- Синтез новых знаний может сопровождаться [[Удовольствие от инсайта]]
	- Поэтому, возможно, внедрение данного этапа в освоение материала будет ялвяться [[Средства для формирования познавательного интереса]]
- Также, данный [[навык]] необходим для внедрения [[Теория системного обучения]], но она же может развивать его
