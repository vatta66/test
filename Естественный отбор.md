# Естественный отбор
![[Естественный отбор-18-06-2024.png|423]]

- Формы естественного отбора [1]
	- [[Движущий отбор]]
	- [[Стабилизирующий отбор]]
	- [[Дизруптивный отбор]] 
	- [[Искусственный отбор]]
	- [[Половой отбор]]

- Основной фактор эволюции, в результате которого в популяции остаются только наиболее приспособленные к среде особи (а точнее [[ген]]ы)
- Естественный отбор сохраняет не хорошие гены
	- А [[ген]]ы, успешно функционирующие в данной среде
		- Среда для генов - другие гены
	- Т.е. сохраняет наиболее [[Приспособленность|приспособленных]] особей
- Выживание приспособленных - частный случай выживания наиболее стабильных
	- Ранняя форма естественного отбора - отбор наиболее стабильных ([[Критерии успешного репликатора]])
