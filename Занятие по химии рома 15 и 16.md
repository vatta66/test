# Урок по химии рома 15 и 16

- Кислоты
	- Получение
		- Прямое сложение компонентов (для бескислородных)
		- к.о + h2o
		- соль + к-та (осадок)
	- Свойства
		- + Ме (до водорода и не активный) = соль + н2
		- + основание = соль + вода
		- + о.о = соль + вода
		- + соль = соль + к-та
			- Закономерность: везде есть металл
	- H2SO3 и H2CO3 сразу распадаются
- Соли
	- Получение![[Pasted image 20230321173439.png]]
	- Свойства
		- + Основание = соль и основание
		- + кислота = соль и кислота (осадок, газ или вода)
		- + Ме = соль + ме (ряд активности)
		- + соль = соль + соль (огв)
