# Классно-урочная форма обучения

- Была придумана [[Я.А.Коменский]]
- Особенности
	- Постоянный состав класса
	- План обучения составляется на основе последовательного изложения материала(уроки)
	- Каждый урок посвящен только одному предмету
- Преймущества:
	- Экономная, т.к. один учитель работает с множеством учеников
	- Создает предпосылки для формирования взаимопомощи, коллективной деятельности, [[Взаимообучение]]
- Главный недостаток
	- Ориентация на среднего ученика
