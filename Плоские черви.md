# Плоские черви
Up: [[Двусторонне-симметричные]]

Тип Plathelminthes, platyhelminthes (плательминтес, платихельминтес)

- Ароморфозы плоских червей
	- Появление двусторонней симметрии
	- Появление [[мезодерма]]
		- Образует паренхиму - основное вещество тела
		- Первичной и вторичной полости нет, только паренхима
	- Появление выделительной системы - [[Протонефридии]]
	- Появление половой системы
	- Появление [[Пищеварительная система]]
	- Кожно-мускульный мешок
		- Сросшаяся покровная и мышечная ткань
- Нет целома
- Нет [[Кровеносная система]]
- Классы
	- Класс [[Ресничные черви]] (Turbellaria)
	- Надкласс Neodermata
		- Паразиты
		- В ходе жизненного цикла происходит образование неодермы (тегумента) (отсюда и название)
			- Когда неодерма проникает в клетку хозяина, она сбрасывает свой эпителий
				- Остается только базальная мембрана
			- На его месте проростает новая ткань [[мезодерма]]льного происхождения с микроворсинками - неодермис
				- Граници между клетками отстутствуют - синцитий
				- По большей части это нужно для более эффективного всасывания питательных веществ через ворсинки
		- Классы
			- [[Трематоды]] (дигенетические сосальщики)
			- Monogenea (моногенетические сосальщики)
				- В основном - [[паразитизм|эктопаразиты]] жабр [[рыбы]]
					- Для этого у моногеней развит прикрепительный диск с присосками и крючьями
				- Сильно развит кишечник
				- Полистома лягушачья
					- Прикрепительный диск с 6 присосками и 2 крючьями
					- [[Жизненный цикл полистомы лягушачьей]]
				- Пример - Diplozoon paradoxum (спайник)
					- В половом процессе сростается с партнером и живет так до конца жизни
			- [[Ленточные черви]] (Cestoda)
- Нет кутикулы
