# Виды перспектив в педагогике коллектива

- Перспектива - [[Целеполагание|цель]], способная сплотить [[коллектив]]([[Этапы формирование детского коллектива]])
- Ближняя
	- Должна опираться на личную заинтересованность ученкиа
		- Каждый воспитанник воспринимает ее как собственную радость и будет обладать [[Внутренняя мотивация]]
			- Объединяя коллектив
	- Например, Близжайший поход в кино, интересная игра и тд
- Средняя 
	- Радостное, несколько отдаленное событие
	- Для его достижения нужно приложить усилия
	- Среднюю перспективу имеет смысл выдвигать, если в классе сформировался [[Этапы формирование детского коллектива|актив]], который сможет организовать коллективную работу по достижению цели
- Дальняя
	- Отодвинутая во времени социально-значимая и требующая значительных усилий цель
- В любой момент времени коллектив должен иметь перед собой цели, на которые можно будет опереться в воспитательной работе
- Непрерывная смена перспектив, их усложнение - обязательное условия развития коллектива
