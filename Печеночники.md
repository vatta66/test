# Печёночники
Up: [[Мохообразные]]
Next:


- Дорзовентральное строение тела: верхняя и нижняя сторона различаются
- Двудомные растения: женские подставки развиваются на одних талломах, мужские - на других
- У некоторых имеются масляные тельца
	- Смесь эфирных масел (возможно для защиты от животных)
- Внутри спорофита есть элатеры - пружинки для рассеивания спор
- На верхней стороне есть центральная жилка (но не из проводящей ткани)
	- На ней расположены меристемальные клетки, которые обеспечивают рост таллома

Строение Маршанции:
![[Маршанция изменчивая.excalidraw]]

- Дихотомически ветвящийся гаметофит (представлен талломом)
	- Основная часть -  паренхима
	- На поверхности (под эпидермой) есть камеры с ассимилирующими нитями - такие столбики из клеток
		- в каждую камеру открывается бочонковидное устье - бочонок из 4 рядов клеток, в каждом по 4 клетки
		- ![[Печеночники-08-06-2025.png|300]]
	- На верхней стороне бывают выводковые корзинки, на дне которых выводковые почки
		- Для вегетативного размножения
	- Ризоиды
		- Простые
			- Заякоривают в субстрате
		- Язычковые
			- Покрыты чешуйками - амфигастриями (они вырастают из таллома и обхватывают ризоиды)
			- Они собраны в пучки, по пространству внутри которого поднимается вода (т.е. не внутри клеток, а между ризоидами)
- ![[Печеночники-09-06-2025 1.png|456]]
	- **А** — воздушная камера, **В** — брюшные чешуи, **С** — ризоиды. Ассимиляционная ткань окрашена фиолетовым
- Строение половых органов
	- Половые органы развиваются на подставках (гаметангиофорах) - вырастах таллома
	- Мужские подставки
		- Немного загнуты кверху, антеридии развиваются на верхней стороне
		- В каждом антеридии из сперматогенных клеток образуются по два двужгутиковых сперматозоида
		- ![[Печеночники-09-06-2025 2.png|343]]
		- ![[Печеночники-09-06-2025 4.png|514]] (4)
	- Женские подставки
		- Загнуты книзу, архегонии развиваются с нижней стороны
		- ![[Печеночники-09-06-2025 3.png|622]]
		- ![[Печеночники-10-08-2025.png|584]]
			- Все архегонии покрыты общим покрывалом - перихецием (п)
			- Вокруг каждого архегония также есть защитный слой клеток - перианций
			- м - молодой архегоний, ст - старый
			- р - ризоиды
		- ![[Печеночники-09-06-2025.png|300]]
- После оплодотворения на месте архегониев под покровом перианция развиваются коробочки на ножке (спорофит)
	- ![[Печеночники-09-06-2025 5.png|300]] к — коробочка, н — короткая ножка, г — гаустория, бр брюшко архегония, п — разросшийся перианций
		- Гаусторией сидит на ткани подставки и питается
	- ![[Печеночники-09-06-2025 9.png|503]]
		-  **А**–**С** — развитие спорангия, разные стадии. **D**–**Е** — зрелый спорангий. **F** — женская подставка с закрытыми зрелыми спорангиями. **G** — открытый спорангий со споровой массой и элатерами. **H** — женская подставка с открытыми спорангиями
	- Внутри коробочки образуются споры с элатерами
		- Споры собраны в тетрады
			- ![[Печеночники-09-06-2025 7.png|218]] - тетрады спор в спорангии
		- У споры есть 2 оболочки: экзоспорий и эндоспорий
- Стадия протонемы очень редуцирована

- Систематика
	- Семейство Маршанциевые (Marchantiaceae)
	- Семейство Коноцефаловые (Conocephalaceae)
		- Представитель - коноцефалум ![[Печеночники-09-06-2025 6.png|300]]
			- Не образует мужских подставок, архегонии развиваются внутри таллома
	- Семейство Риччиевые (Ricciaceae)
		- Водное растение
		- Вообще не образуют подставок, все развивается внутри таллома
		- ![[Печеночники-09-06-2025 8.png|300]]
