# Основные эпидермальные клетки
Up: [[Эпидерма]]
Next:

- Есть у всех
- В них содержатся только [[Пластиды|лейкопласты]]
- По форме может быть разной
	- Часто встречается вариант, когда основные клетки извилистые, и подходят друг к другу как пазлы - это позволяет им прочнее сцепляться
		- ![[Основные эпидермальные клетки-07-08-2025 2.png|300]]
	- Клетки могут быть длинными вытянутыми (часто у однодольных)
	- Иногда по форме основных клеток можно определить растение
	- ![[Основные эпидермальные клетки-07-08-2025 1.png|300]]
- Вторичная клеточная стенка с большим количеством простых пор
	- При чем часть клеточной стенки, которая граничит с поверхностью, всегда толще остальных
	- ![[Основные эпидермальные клетки-07-08-2025.png|300]]
- Характерно наличие кутикулы
	- Всегда есть на той стороне, которая соприкасается с воздухом
	- Рекорд по размеру кутикулярного слоя у пальмы copernicia cerifera (бразильская восконосная пальма) - до 2 см
		- Из нее делают карнаубский воск
	- Также, кутикула может помогать фокусировать солнечный свет, если его в окр среде мало (в тропиках)
		- Пример - ![[Эпидерма-07-08-2025.png|300]] mapania caudata
- Может быть кутинизация
	- Т.е. кутин есть не только на поверхности в составе кутикулы, а сама [[Клеточная стенка]] пропитана им
	- Часто есть у пустынных растений, [[Суккуленты]]
- Может происходить минерализация клеточных стенок
	- Очень часто с помощью оксида кремния
	- Еще бывает отложение карбоната кальция, щавелевокислой извести, пектиновокислой извести
		- Отложение карбоната кальция - это один из вариантов удаления излишков кальция из организма
