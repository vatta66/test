# Микронуклеус [[Инфузории]]

- хранение наследственной информации
- Не используется для [[Синтез белка]]
- Только для полового процесса - [[Конъюгация инфузорий]]
- В нем находится очень много [[Транспозоны]]
	- Участки [[днк]] находятся в хаотичном порядке, постоянно перемешиваются
		- Последовательности [[ген]]ов разрушены
		- Для того, чтобы сделать днк пригодной для считывания, нужно собрать все участки в единые гены
			- ДНК макронуклеуса используется как матрица для синтеза РНК (см. конъюгацию инфузорий)
			- Эта РНК попадает сначала в цитоплазму, а потом в новый макронуклеус (т.к. старый уничтожается, см. выше)
				- В нем есть хромосомы из микронуклеуса, гены в которых перепутаны
			- хромосомы микронуклеуса перестраиваются согласно РНК
			- Происходит добавление [[теломеры]]
