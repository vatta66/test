# Нервная система лестничного типа

- Головной ганглий (церебральный ганглий)
	- Надглоточный ганглий
	- Подглоточный ганглий
	- Соединяются между собой, образуя окологлоточное нервное кольцо
		- Сквозь это кольцо проходит канал [[Пищеварительная система]]
- Два ряда продольных нервных тяжей
	- Образуют брюшную нервная цепочка
	- Они соединены между собой
		- Комиссурами
			- Поперечными нервными стволами
		- Коннективами
			- Продольными нервными стволами
	- Иногда могут соединяться в один 
- характерна для [[Плоские черви]]
