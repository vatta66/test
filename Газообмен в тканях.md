# Газообмен в тканях

- Клетки нуждаются в кислороде, необходимом для тканевого дыхания (для аэробного этапа - [[Энергетический обмен клетки#^53d8bb]]), а углекислый газ, побочный продукт обмена веществ, требует удаления из клетки в [[Кровь]] 
	- Углекислый газ образуется в процессе [[Энергетический обмен клетки]]
- Движущей силой является разность парциальных давлений газов 
	- Вещества устремляются из области большего давления в область меньшего
		- Пример: парциальное давление кислорода в альвеоле 100-110, а в венозной крови [[Капилляры]] , оплетающего стенку альвеолы, давление кислорода 40. Таким образом, кислород устремляется из области большего давления в область меньшего - из альвеолы в кровь.
