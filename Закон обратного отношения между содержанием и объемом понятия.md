# Закон обратного отношения между содержанием и объемом понятия

- Если первое понятие шире другого по [[Объем понятия в логике]], то оно беднее его по [[содержание понятия]] (совок отличительных черт данного понятия)
	- Если же первое понятие уже второго по объему, то оно богаче его по содержанию
- Закон применим только при вхождении объёма одного понятия в объём другого
