# Инстинкт

- Фазы инстинкта:
	- Поисковое поведение
		- Поиск ситуации, способной удовлетворить соответствующую потребность. 
			- Может происходить из-за
				- [[Снижение порогового значения запуска инстинкта]]
				- Раздражения рецепторов, [[Гормоны]] и тд
			- Может изменяться в зависимости от опыта субъекта
	- Завершающий(консумматорный) акт
		- Детерминирован [[ген]]етически
		- Элементы этого акта навзаны Фиксированным комплексом действий(ФКД)
			- Эти действия являются
				- Врожденными(не требуют обучения, тренировки), Шаблонными, Видоспецифичными(характерны для данного вида)
					- Благодаря этим характеристикам [[элементы консумматорного акта могут выполнять роль таксономических признаков]]
