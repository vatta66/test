# Хламидамонада

![[Хламидомонада-11-07-2024.png]]


- Один из родов [[Зеленые водоросли]]
- 2 жгутика для передвижения
- Имеет [[стигма]]
- В [[Клеточная стенка]] нет [[целлюлоза]]
- Может являться пищей для [[Амебы]]
- ![[Pasted image 20240929212033.png]]
	- Бесполое: клетка функционирует как зооспорангий, внутри митозом образуются зооспоры
	- Половой процесс: изогамия, гетерогамия, оогамия
		- +гаметы и -гаметы встречаются передними концами, скручивают жгутики и сливаются в зиготу
		- Зигота покрывается оболочкой и становится зигоспорой
		- Весной зигоспора проходит через мейоз с образованием зооспор
- Снежная хламидамонада, может окрашивать снег в розовый цвет
	- ![[Pasted image 20230919161731.png|300]]![[Pasted image 20231019222058.png||100]]
	- Имеет пигмент [[Астаксантин]], который и придает ей розовый цвет
	- Снег хорошо отражает свет благодаря белой окраске, но снежные хламидомонады делают его розовым, понижая его способность к отражению, из за чего он начинает таять быстрее. Ускоряется и таяние ледников - одна из [[Глобальные проблемы]]
- При делении хламидамонады между двумя дочерними клетками может образоваться цитоплазматический мостик
	- Обычно он покрывается [[Хитин]]овым слоем
	- Через него будет происходить обмен пит. в-вами
	- Таким образом, хламидамонады могут образовывать довольно большие колонии (с помощью этих мостиков)
		- В таком случае часть хламидамонад становится стерильными (не размножаются)
		- Примером такого колониального организма может служить [[Вольвокс]] (volvox) ^4fe7e0
