# Устьица

- Виды
	- Самый простой тип - аномоцитный
		- Две замыкающие клетки
		- Между ними - замыкающая щель (аппертура устьица)
			- Является единственным межклетником [[Эпидерма]], но его размер можно регулировать
	- Плавающие
		- Замыкающие клетки окружены одной большой побочной клеткой ![[Устьица-01-09-2025.png|356]]
	- Диацитный
		- Каждая замыкающая клетка окружена двумя побочными ![[Устьица-01-09-2025 1.png|292]]
	- Триацитный
		- Замыкающие клетки окружены тремя побочными  ![[Устьица-01-09-2025 2.png|300]]
	- Анизоцитный тип
		- Замыкающие клетки окружают три побочные, но они отличаются по размеру ![[Устьица-01-09-2025 3.png|300]]
	- Тетрацитный
		- ![[Устьица-01-09-2025 4.png|678]]
	- Граминальные
		- Замыкающие клетки в виде гантель ![[Устьица-01-09-2025 5.png|300]]
	- С побочными клетками
		- Они обеспечивают энергией замыкающие клетки
		- Имеют крупные вакуоли - вода для замыкающих клеток
- Образование устьиц
	- Мезогенный способ
		- Все клетки устьичного аппарата формируются из одной клетки протодермы
	- Перигенный
		- Замыкающие клетки формируются из 1 клетки протодермы, побочные клетки - из другой/других
	- Мезоперигенный
		- 2 замыкающие и 1 побочная формируются из 1 клетки протодермы, остальные побочные - из других
- Замыкающие клетки
	- Вторичная оболочка неравномерно утолщенная
	- Есть только хлоропласты
	- Есть кутикула
		- Для того, чтобы в них не падал тургор и они могли нормально работать
- Работа устьиц
	- В замыкающие клетки против градиента концентрации из побочных или основных клеток переходят ионы калия
		- Энергию для этого как раз вырабатывают хлоропласты
	- Вслед за калием по осмосу в замыкающие клетки из всех соседних начинает переходить вода
		- Их боковые стенки очень плотные, поэтому клетка начинает расширяться в направлении “верх низ”,  но так как вторичная клеточная стенка не может бесконечно растягиваться, за верхними и нижними сторонами подтягиваются и боковые - устьице открывается
	- Когда вода выходит, устьице закрывается
	- ![[Устьица-08-08-2025.png|663]]

- Через них осуществляется газообмен ([[Функции листа]])
