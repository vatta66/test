# Анапсидный vs Синапсидный vs Диапсидный череп
Up: [[Амниоты]]
Next:

- Анапсидный
	- Нет отверстий за глазницами
- Синапсидный
	- характерен для синапсид - линии, давшей [[млекопитающие]]
	- Есть одно отверстие за глазницами для крепления мышц челюсти
- Диапсидный
	- За глазницами по два отверстия
	- Диапсиды - линия, давшая [[Пресмыкающиеся]]
