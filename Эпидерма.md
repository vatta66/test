# Эпидерма
Up: [[Покровная ткань растения]]
Next:

- Первичная покровная ткань побега и репродуктивных органов
- Образуется из [[Апикальная меристема|протодермы]]
- У большинства однослойная
- Это сложная ткань, в ней есть несколько типов клеток
	- [[Основные эпидермальные клетки]]
	- [[Устьица|Клетки устьичного аппарата]]
	- [[Гидатоды]] (водяные устьица)
		- есть еще гидатоды, выделяющие капельно-жидкую воду, но это уже секреторно-*выделительная* система
	- [[Трихомы]]
	- [[Эмергенцы]]
		- не полностью клетка эпидермы
	- Околоволосковые клетки
		- У некоторых растений в основании трихом есть такие клетки, которые укрепляют трихомы 
	- Гидраподы
		- у растений, живущих в воде
		- Нужны для того, чтобы всасывать растворы из воды
		- Скорее всего не относится к эпидерме, т.к. отличается по функциям, но развивается все еще из протодермы
			- Поэтому гидраподы - это идиобласты в составе эпидермы
- [[Эпидермис листа]]
