# Ошибка выжившего

- One of the [[Когнитивные искажения]]
- We have a lot of data about “survivors”, but have very small about “the dead”
	- Выжившие могут быть из за удачи
- Тенденция обращать [[внимание]] только на истории успеха, создающая искажённую картину, игнорирующую неудачников и выбывших
	- It might be because of the [[яркость образа]] of those who won
		- Because of it these images easily comes to our mind, and using [[Эвристика доступности|availability heuristic]] we consider them as more true
	- Also people neglect the amount of winners and losers, so its a [[Нечувствительность к размеру выборки]]
- This cognitive distortion, as each of them, distort [[Принятие решения]]
- Из старых зданий остаются только самые красивые и прочные — только потому, что остальные сносят, а не реставрируют. Из техники старых поколений остаются только те образцы, которые сделаны хорошо
