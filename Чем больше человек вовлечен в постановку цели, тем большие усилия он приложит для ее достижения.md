# Чем больше человек вовлечен в постановку [[Целеполагание|цель]], тем большие усилия он приложит для ее достижения

- Тезис [[Теория постановки целей Локка]], объяснение - мое
- Это можно объяснить с помощью теории [[Когнитивный диссонанс]]
	- Если я участвовал в постановке цели, значит она важна для меня, значит я должен приложить большие усилия для ее достижения
	- Главное не заставлять людей, иначе произойдет [[Редукция когнитивного диссонанса]]
- Как результат происходит увеличение [[Внутренняя мотивация]] на достижение цели
- Возможно, что [[коллектив]], лидер(или актив) которого организует всех участников в процессе целеполагания, сможет добиться большей слаженности коллектива в выполнении задач, что позволит быстрее перейти на новый этап - [[Этапы формирование детского коллектива]]
- Можно использовать в [[Педагогика MOC]]
	- В начале урока/недели/месяца вместе с учениками ставить план - они больше будут вовлечены в достижение цели - [[Целеполагание]]
