# Пенсии
Up:
Next:

- **Пенсия** - ежемесечная [[деньги|денежная]] выплата в целях компенсации застрахованным лицам [[Заработная плата|заработной платы]] и иных выплат, утраченных с наступлением нетрудоспособности по старости или инвалидности, или по случаю потери кормильца
- Виды пенсий
	- По потере кормильца
	- По инвалидности
	- По старости
- Страховые пенсии говорят о россии как о [[Федеративное устройство РФ#^8dd492|социальном государстве]]
