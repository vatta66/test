# Эвристики

27-04-2023
- Method helps you to [[Интуиция|intuitively]] give an answer on a hard question
- When there is no quick answer on a difficult question, you [[Система 1]] replace this question with an easier one and give you an answer on it
	- And you think that this answer is the answer on the first question
	- This second question is an [[Эвристики|heuristical]] [[вопрос]]
- Also you can use it deliberately with your [[Система 2]]
	- When you cant solve some problem ([[Решение проблемы]]), you can replace it with similar one and solve it - [[Решение задачи через аналогичную]]
