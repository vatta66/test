# Способы системного решения проблем

[[Эффективное мышление MOC]]

При [[Решение проблемы]] нужно представить ее в виде системы, определить составляющие этой системы и связи между ними

Формирование целосного понимания проблемы:
1. **Круг связей**: 
	1. Рисуем круг
	2. За пределами круга приклейте стикеры и напишите на них все ключевые факторы, элементы и события, а также всех ключевых людей, которые задействованы в изучаемой проблеме.
	3. Начните изображать выявленные взаимосвязи между элементами, рисуя линии, связывающие элементы вместе. 
	4. Можно использовать стрелки и знаки «+» и «–» для отображения характера взаимосвязей между элеменатами
	5. Определите, к каким элементам ведёт много линий. Точки с большим количеством связей могут потенциально сильно влиять на многие другие. Их называют **точками приложения усилий** (точки опоры). Скорее всего, ваше решение будет связано с воздействием именно на эти точки
	6. Выбрать две точки приложения усилий, для каждой из них придумать решение.
2. **Круг мнений**: берем ту же проблему, рисуем круг, обозначаем на нем всех, у кого есть интерес к данной проблеме. Далее выясняем, как можно связаться с каждой из сторон, узнать их мнение. Далее анализируем полученные данные
3. **Обозначение контекста**:  рисуем график, на оси y обозначаем изменяющиеся факторы(факторы, вовлеченные в проблему), на оси x - прошлое, настоящее, и будущее. Данные прошлого и настоящего берем из интернета, данные будущего прогнозируем сами. Суть в том, что на одном графике вы сможете увидеть изменение различных факторов, влияющих на проблему, найти взаимосвязи, сформировать контекст происходящей проблемы.  ![[Pasted image 20220424182542.png]] 
4. 

Реализация решения проблемы:
1. Продумывание возможных вариантов будущего: **квадрант мнений**: рисуем квадрат, по одной оси пишем одно решение(см. круг связей п. 6), по другой - второе. Расставляем плюс и минус на концах каждой линии. Далее в каждую ячейку пишем все последствия, которые произойдут при данных условиях ![[Pasted image 20220424184144.png]]
2. **SCAMPER**: 
	1. S - заменить
	3. C - комбинировать
	4. A - адаптировать
	5. M - модифицировать
	6. P - использовать
	7. E - исключить
	8. R - преобразовать
3. **Воронка осуществимости**
	1. У нас достаточно времени на реализацию?
	2. У нас достаточно экспертизы, мы сможем это реализовать?
	3. Эта идея масштабируется?
4. **Эффект кругов на воде**: нарисовать несколько кругов: первый - ваше решение, на последующих - события, к которым оно приведет. Каждый последующий круг вытекает из предыдущего. Пишите как позитивные, так и негативные последствия
