# Двустворчатые моллюски
Класс Bivalvia < тип [[Моллюски]]
[[Двустворчатый моллюск беззубка.excalidraw]]


- Мантия в виде двух листков
	- Наружный и внутренний
	- На заднем конце тела образуют два отверстия - сифоны
		- В вводной сифон вода с веществами попадает
			- На его стенках есть кутикулярные структуры для фильтрации крупных частиц
		- Из выводного выходит
			- Из него также выводятся гаметы и глохидии (если есть)
		- У прогрессивных форм сифоны могут быть очень длинными
			- Сам моллюск зарывается в грунт и выставляет сифон наружу
			- Пример – гуидак ![[Двустворчатые моллюски-17-05-2025 1.png|600]]
- Две створки раковины
	- Есть мускулы-замыкатели
		- 1-2 штуки по бокам тела
	- Есть замок
		- Совокупность зубцов на одной створке и соответствующих углублений на другой
		- Виды
			- Таксодонтный
				- Все зубцы одинаковые, примитивный вариант
			- Гетеродонтный
				- Зубцы разные, прогрессивный вариант
	- Есть лигамент
		- Органическая связка (из белка)
		- Производное периостракума
	- ![[Двустворчатые моллюски-17-05-2025.png|824]]
	- Летом раковина растет быстро, зимой - медленно, поэтому появляются годичные кольца - по ним можно определить возраст моллюска
- Голова редуцирована из за малоподвижного образа жизни
	- От нее остается 2 пары ротовых лопостей
	- Это - [[общая дегенерация]]
- Нога есть не всегда
	- Если есть, то перемещение очень медленное
	- Чаще служит для прикрепления
	- В районе есть биссусные железы
		- Биссус - очень прочное вещество
			- Поэтому двустворчатых (например, мидий), очень сложно оторвать от субстрата
			- Белки нитей биссуса активно используются в медицине, фармацевтике, косметологии, в качестве хорошего адгезивного агента или же покрытия, предотвращающего обрастание различных поверхностей
- [[Пищеварительная система]]
	- Ротовые лопасти → пищевод → желудок с протоком печени → длинная средняя кишка с петлями → задняяя кишка
		- Большая часть средней кишки находится в ноге
		- Задняя кишка проходит сквозь желудочек сердца
	- В нем есть кристаллический стебелек
		- Стебелек из ферментов
		- На него наматывается слизистый тяж пищи и переваривается
		- Работают по принципу ступки и пестика
			- Растирают, размельчают пищу
	- Кишка проходит через сердце
- Выделительная система
	- Почки, собирают продукты из перикарда
	- Кеберов орган
		- Сильно разветвленная стенка перикарда
		- Служит для накопления продуктов обмена
			- Потом выводит их в полость перикарда
- Дыхательная система
	- Чаще всего - пластинчатые жабры
- Половая система
	- Раздельнополые
	- Гонады чаще всего находятся в ноге
- ![[Двустворчатые моллюски-17-05-2025 3.png|599]]
- Нервная система ганглиозного типа
	- Часть ганглиев сливаются друг с другом, образуя синганглий
		- В передней части тела - цереброплевральный синганглий
			- Иннервирует ротовые лопости, переднюю часть пищеварительной системы и передний мускул-замыкатель
			- От него тяж идет на следующий синганглий
		- Висцеропариетальный синганглий
			- Иннервирует сифоны, задний мускул-замыкатель и большую часть органов
	- Педальный ганглий
		- Иннервирует бисусную железу и мускулатуру ноги
- В их дивертикулах способны накапливаться [[токсины]] вещества
	- Когда происходят [[красные приливы]], они могут съедать [[Динофлагеллята]]
		- Моллюск становится ядовит для человека
		- Моллюски лучше всех накапливают эти токсины
- Личинки бывают двух типов
	- Велигер
		- Есть зачаток ноги и зачаток раковины
			- Способна к активному плаванию
			- С помощью этой стадии происходит расселение
		- Есть парус - дорсальный вырост с ресничками
	- Глохидий
		- Паразитическая личинка (характерно для беззубки)
		- Когда [[Рыбы]] хочет съесть двустворку, она выбрасывает в нее облако глохидий
			- Они приклепляются створками к ее жабрам
				- Используют рыб для расселения
		- При интенсивном вылове рыб численность популяций беззубки может сокращаться, так как личинкам будет негде развиваться
		- ![[Pasted image 20230719184438.png]]
			- Каждая створка имеет длинный первичный зубец с маленькими вторичными зубцами - для прикрепления к хозяину
			- Один мускул-замыкатель
		- ![[Двустворчатые моллюски-05-07-2025.png|608]]
- Классификация
	- надотрад Protobranchia (первичножаберные)
		- Имеют примитивные ктенидии (не способны к фильтрации, только газообмен)
		- род Nucula
			- Собирают пищу с помощью гипертрофированных ротовых лопостей
	- надотряд Filibranchia (нитежаберные)
		- *Ктенидии удлинняются и закручиваются - увеличиваются в размере*
			- Теперь могут еще и фильтровать воду
		- отряд Mytilida (мидии)
		- отряд Ostreida (устрицы)
		- отряд Pectinida (гребешки)
			- У них есть до 200 глаз (синие) ![[Двустворчатые моллюски-13-11-2025.png|422]]
				- В них в качестве фокусирующего элемента - вогнутое зеркало, из примерно 20 слоев “черепиц” [[гуанин]]а ![[Двустворчатые моллюски-13-11-2025 1.png|537]]
			- Реактивный способ передвижения за счет схлопывания створок раковины
		- отряд Arcoida
	- надотряд Eulamellibranchia (перегородчатожаберные)
		- *Формируются поперечные перегородки между листками ктенидия*
			- Пластинчатые жабры с ресничным эпителием
		- отряд Uniomida
			- Беззубка, перловица
		- отряд Venerida
			- Самый крупный отряд двустворчатых
			- Карабельные черви
				- Раковина почти редуцирована, остался только небольшой кусок на переднем конце тела
					- С помощью него проделывает дыры в древесине
			- Kuphus polythalamia
				- Может достигать длины полутора метров
				- Живут в трубках
	- надотряд Septibranchia
		- Почти полная редукция ктенидий
			- не газообменивают и не фильтруют - моллюск дышит всей поверхностью тела
			- Оставшиеся две небольшие части могут сдвигаться и раздвигаться, тем самым засасывая воду с пищей в мантийную полость
		- род Poromia
		- род Cuspidaria
