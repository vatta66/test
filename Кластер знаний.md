# Кластер знаний
14.1.24:
- Уровень вложенности кластеров знаний может меняться
	- Так, кластер по биологии в 5 классе будет выглядеть как в примере выше, но со временем некогда атомарное знание о том, что тело состоит из клеток, превратиться в кластер знаний “Клетка человека”, который будет состоять из сввоих атомарных знаниях о составляющих клетки. Эти атомарные знания со временем также превратятся в кластеры, и так далее
- Система [[Атомарное знание|атомарных знаний]], построенная вокруг одной темы
	- Например, клстер знаний по биологии, психологии и тд
- Таким образом, любая тема, имеющая достаточное количество информации для связи [[Атомарное знание|атомарных знаний]] этой темы друг с другом, становится кластером знаний
- В процессе формирования у знаний, составляющий этот кластер, появляется [[смысл]] 
	- [[Смысл - эмерджентное свойство системы знаний]]
		- Т.е. смысл присущ только кластеру как системе, а не его частям (атомарным знаниям) по отдельности
- Без построения кластера знаний невозможно создать [[Сеть знаний]]
- Создать кластер знаний можно с помощью [[Фреймворк изучения темы]], [[Обучение по слоям]], [[Системная спираль изложения]]
- На этапе создания кластера начинается действие эффекта [[Искусственная актуализация знания]]
