# Снижение порогового значения запуска инстинкта

- [[Конрад Лоренц]] думает, что
- Чем дольше не проявляется инстинктивное поведение
	- Тем сильнее снижается порог запускающего его раздражителя
		- Порог может снизится до нуля, а значит инстинктивное поведение может запустится без внешнего стимула
	- Тем сильнее животное ищет стимул для его запуска
- Так, при долгом сдерживании, [[Агрессия]] может вырваться по самым пустяковым причинам
	- По этой логике должна работать [[Теория катарсиса]], но, как показывают эксперименты(внутри ссылки), это не так
