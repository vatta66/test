# Урок по химии Рома 14

- Основания
	- Растворимые (щелочи, основания)
		- 1А, 2А
		- Сильные
	- Нерастворимые
		- Слабые
- Все - твердые
- Получение
	- Щелочи
		- Me + H2O = MeOH + h2
		- MeO + H2O = MeOH
	- Нерастворимые
		- Р. к-та металла  + щелочь
- Свойства
	- Кислота + гидроксид = соль + вода
	- Щелочь + кислотный/амфотерный оксид = соль + вода
	- Нераств осн = оксид + вода
