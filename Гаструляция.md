# Гаструляция

- Часть бластулы как бы впячивается (как если бы вы вдавили кулаком в мяч)
- Гастроцель - углубление, которое образовалось
	- Первичная пищеварительная полость
		- Из нее разовьется [[Пищеварительная система]]
- Часть, соединяющая гастроцель с окружающей средой - первичный рот
- Образуются 
	- [[эктодерма]]
		- Те клетки, которые остались снаружи
	- [[Энтодерма]]
		- Клетки, образующие первичную пищеварительную полость
