# Работа флоэмы
Up: [[Флоэма]]
Next:

- Клетки спутницы из исходного органа (который производил сахара, чаще всего [[лист]]ья) загружают сахара во флоэму
- Из за этого по осмосу вода из сосудов [[Ксилема]] начинает переходить в эту ситовидную трубку
- Снизу этой трубки целевой орган забирает из нее сахара
	- Опять же с помощью клеток-спутниц
	- Из за этого осмотическое давление падает, и вода переходит обратно в ксилему, которая опять доставляет ее наверх
- Получается, что наверху много воды, а снизу - мало
	- Это создает давление на воду, поэтому она перемещается снизу вверх
