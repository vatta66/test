# Модульное(программированное) обучение

- Основная цель
	- Сделать обучение максимально контролируемым - отслеживание каждой стадии
		- Для оказания ученику своевременной помощи
- Основоположниками были Н. Краудер, [[Б. Скиннер]], С. Пресси
- Особенности ПО:
	- Учебный материал разделяется на модули
	- Учебный процесс состоит из последовательного усвоения модулей
	- После каждого модуля следует контроль
		- При неправильном ответе ученик получает помощь
		- При правильном - продвижение дальше
	- Каждый учащийся работает самостоятельно и в удобном для него темпе
	- Педагог выступает в качестве организатора, консультанта, контролера, осуществляет индивидуальный подход
- Может быть результативно сочитать традиционный способ обучения с модульным
