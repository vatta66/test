# Положительное [[подкрепление]]

- Нечто желанное для субъекта, что будет для него наградой, тем самым становясь источником [[мотивация]] для такого же действия в будущем
- Положительное и [[Условное положительное подкрепление]] эффективно тогда, когда нужно выработать новое поведение
	- Для отмены нежелательного поведения стоит воспользоваться [[Отрицательное подкрепление]]
- Чтобы нечто стало положительным подкреплением, субъект должен желать этого
	- Пища не является положительным подкреплением для сытого человека, улыбка и похвала будут бесполезны, если человек пытается вывести вас из себя
		- То есть, у человека должна быть [[Потребность]] в чем-то, что станет подкреплением
	- Чтобы субъект возжелал что-либо нейтральное, можно воспользоваться [[Условное положительное подкрепление]]
- Положительное подкрепление работает засчет работы [[Система поощрения]] 
	- Эволюционно-выработанной системой положительного подкрепления является [[дофаминовое подкрепление]]
- Чем меньше положительное подкрепление, которого хватает для достижения результата, тем лучше
- Постоянное положительное подкрепление необходимо только на этапах [[Нейробиология обучения|обучения]]
	- Когда поведение надежно закрепилось, можно использовать [[Вариативное подкрепление]]
- Положительное подкрепление удобно использовать совместно с эффектом [[угасание]]
