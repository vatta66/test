# (не)Будь собой
Up:
Next:

- В выражение “будь собой!” и все советы по типу “нужно не строить из себя непойми кого, а быть тем, кем ты на самом деле являешься” нужно добавить одну важную деталь: “будь собой, если тебе нравится то, какой ты есть”
	- Т.е. есть ли у тебя [[Любовь к себе]]
- Например, такие советы применимы для [[отношения]], [[общение]]
- Если тебе не нравится то, какой ты есть, совершенно нормально “строить” из себя того, кем ты не являешься - fake it till you make it, чтобы [[решение проблемы|решить проблему]], которая у тебя есть
	- Для определения этого вам поможет [[рефлексия]]
