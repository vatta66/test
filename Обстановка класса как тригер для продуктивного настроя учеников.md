- Ваш класс должен стать триггером для учеников -  [[Привычка изменять психологическое состояние после триггера]] - с помощью него они должны входить в состояние [[продуктивность]], [[Дисциплина|дисциплинированности]], сосредоточенности, [[внимание|внимательности]]
