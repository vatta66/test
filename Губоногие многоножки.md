# класс Chilopoda
Up: [[Многоножки]]
Next:

из группы Opistogoneata 

- хищники
- Тагмозис
	- Строение придатков головы
		- Верхняя губа
			- Из двух пластинок, соединенных треугольным склеритом
			- Есть щетинки - сенсорная функция
		- Мандибулы
			- Небольшие с щетинками
		- Максиллы 1
			- Коксоподит, от него небольшой эндоподит и двучленистый экзоподит с щетинками - щупик
		- Максиллы 2
			- Коксоподиты двух максилл и стернит сливаются – коксостернит
			- От него отходят два щупика из 3 сегментов с шипом на конце
		- ![[Многоножки-24-04-2025.png|300]]
	- Туловище
		- Первый сегмент маленький
		- Несет большую ногочелюсть с ядовитой железой
			- Слившийся коксостернит
			- От него отходит вертлуг 1 (маленький), вертлуг 2 (большой)
			- Затем - 3 сегмента: бедро, голень, лапка (длиннее)
			- ![[Многоножки-24-04-2025 1.png|300]]
		- Чередование длинных и коротких сегментов
			- Увеличивает подвижность тела
			- Строение конечности
				- Тазик → вертлуг 1 → вертлуг 2 → бедро → голень → лапка → предлапка → коготок
				- ![[Многоножки-24-04-2025 2.png|300]]
		- Последняя сегмент с конечностями несет длинные чувствующие ноги
			- Могут трясти ими и издавать с их помощью звуки - для защиты, для осязания
		- Предпоследний сегмент - прегенетальный
			- С короткими гоноподами
		- Последний сегмент - генетальный
			- С гонопором - половым отверстием
