# Жизненный цикл полистомы лягушачьей
Up: [[Жизненные циклы сосальщиков]]
Next:

- Живут в мочевом пузыре лягушки
	- К половому размножению переходят тогда, когда к нему перейдет лягушка
	- ![[Жизненный цикл полистомы лягушачьей-20-12-2024.png|299]]
- Яйца имеют выросты, которыми держатся за растительность в воде
- Из яйца выходит личинка - онкомирацидий
	- У нее будет прикрепительный диск с крючками (без присосок)
	- Покрыта временным ресничным покровом, есть рецепторы
- Прикрепляются к наружным жабрам молодых головастиков
	- Линяют
- Развиваются в половозрелых особей, копулируют
	- Яйцо такое же, как и раньше
	- Из него развивается онкомирацидий
- Онкомирацидий находит себе взрослого головастика
	- Ползут по покровам в анальное отверстие, через него попадают в мочевой пузырь
	- Там живет все время, пока лягушка не станет половозрелой
