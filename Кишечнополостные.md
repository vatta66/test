# Кишечнополостные
Правильное название - Radiata (Diblastica)

- Ароморфозы кишечнополостных
	- Появление многоклеточности
	- Появление дифференциация клеток
	- Появление нервной системы диффузного типа
	- Появление радиальная симметрия
	- Появление [[эктодерма]] и [[энтодерма]] (двуслойность)
	- Появление полостного пищеварения
		- Клетки образуют полость - замкнутая [[Пищеварительная система]] (есть только рот)
- Много рецепторов
	- хеморецепторы (запускают [[Потенциал действия]]) в нервной системе
	- Механорецепторы
		- Статоцисты - [[Вестибулярный аппарат]]
			- Шар из клеток с микроворсинками, внутри которого находится какой то твердый предмет
				- Принцип работы такой же, как и у человека (см. по ссылке)
	- Фоторецепторы
		- Простые структуры
			- Реснички с фоточувствительными белками
		- Глазной бокал
			- Углубление, внутри которого находятся реснички с фоточувствительными белками
				- Позволяет определять, с какой стороны поступил свет
					- Т.к. одни рецепторы будут докладывать о темноте, а другие, на которые попал квант света - о свете
		- Глазной бокал с хрусталиком
			- Может давать больше информации об объекте благодаря фокусировке
- Типы
	- [[Книдарии|Cnidaria]]
	- Гребневики (Ctenophora)
		- Не образуют колоний
		- Почти все тело - мезоглея
		- Два длинных щупальца
			- Для ловли добычи
			- Может полностью убрать их внутрь тела
				- В щупальцевые влагалища
		- Поедают [[Книдарии]] и встраивают [[Стрекательная клетка]] себе
		- Есть два анальных отверстия
