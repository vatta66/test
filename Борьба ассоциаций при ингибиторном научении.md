# Борьба ассоциаций при ингибиторном научении

	- В ходе [[Экспозиция в психотерапии]] происходит работа с данным объектом в различных ситуациях
	- Согласно основной [[Задача экспозиции]] и исходя из [[Теория ингибиторного научения]], это нужно для того, чтобы человек оценивал объект как неопасный (новая ассоциация - [[Ассоциативное обучение]]) в как можно большем количестве случаев
- Возможно, имеет место быть [[Закон Йоста]]
