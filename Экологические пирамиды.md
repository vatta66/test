# Экологические пирамиды
Up: [[Экология]]
Next:

Самый нижний слой - продуценты

- Пирамида численности
	- Отражает количество особей на каждом уровне
	- Обычно, количество особей уменьшается с каждым уровнем
	- Перевернутая пирамида характерна для леса: количество [[насекомые]], которые питаются деревьями, больше этих деревьев
- Пирамида биомассы
	- Обычная пирамида характерна для суши
	- Перевернутая характерна для океанов: фитопланктона по массе меньше, чем зоопланктона
		- Т.к. фито быстро умирает, и его ест зоопланктон
