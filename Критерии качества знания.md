# Критерии качества знания
По Паламарчуку ([[Школа учит мыслить Паламарчук]])

- На основе этих уровней можно выделить [[Три уровня знаний по Паламарчуку]]

- Необходимыми и достаточными для проверки и оценки являются такие качества знаний: 
- Правильность
	- Степень соответствия эталону
- Полнота
	- Объем знаний, соответствующий эталонному
- Умение изменять
	- Применять к знаниям [[Мыслительные операции]]
- Действенность
	- Умение применять знания на практике
		- [[Практический метод в педагогике]]
- Систем­ность
	- Наличие [[Кластер знаний]]
