# Эффект разделения внимания

- Effect occurs when person split their [[внимание]] between several types of information
	- Even if this information was placed here to explain each other
		- Ex. some graphical material and text explaining it
			- Persons attention split between text and graph and their [[Когнитивная нагрузка|cognitive load]] become filled
		-  Also this effect can occur through visual and auditory splits
			- Important thing for [[Визуализация для освобождения ресурса мозга]] is to avoid split attention effect
	- [[Cognitive Load Theory and the Format of Instruction Annotations]]
- Also when two pieces of information that should be connected presents separately in time
- To avoid this effect you should 
	- Integrate text into visual information and so on, but dont place them near to each other
	- Replace some part of [[визуальное представление информации]] by audial one ^2eca20
		- It might be because hearing channel has its own [[Рабочая память]] capacity - [[Вопросы для изучения]]
		- обучающиеся учатся более эффективно, когда им показывают визуальную информацию, сопровождаемую повествованием, а не ту же визуальную информацию с добавленным текстом на экране [2]
