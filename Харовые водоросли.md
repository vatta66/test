# Харовые водоросли
Up: [[Водоросли]]

![[Зеленые водоросли#^ae6abb]]


-  Классы
	- класс Конъюгаты (Zygnemophyceae)
		- Коккоидный и нитчатый тип таллома
		- Полное отсутствие жгутиковых стадий
		- Половой процесс - конъюгация
			- Виды
				- Лестничная - между клетками двух нитей (у клеток должны быть разные знаки - + и -)
				- Боковая - между соседними клетками одной нити
		- Порядки
			- Десмидиевые (Desmidiales)
				- Коккоидный тип таллома
				- Клетка из двух половинок, у каждой своя клеточная стенка
					- Получается выраженная двусторонняя симметрия
				- В оболочке есть поры, через которые выделяется слизь, за счет чего идет движение водоросли
				- ![[Харовые водоросли-21-06-2025 1.png|220]]
				- Представители: пениум, клостериум (не имеют перетяжки)
			- Зигнемовые (Zygnematales)
				- [[Спирогира]]
	- класс Харовые (Charophycea)
		- Внешне напоминают хвощи благодаря мутовчатому строению
			- ![[Харовые водоросли-21-06-2025.png|229]]
		- Поверхность главной оси покрыта защитным слоем клеток (кора)
		- Размножение
			- Вегетативное
				- Клубеньками, которые образуются на ризоидах
				- Ветвями, которые укореняются и растут в грунте
			- Половое - оогамия
				- Ж.ц. обычный, с ооспорой
				- Антеридии и оогонии многоклеточные - единственные, у кого так
			- Бесполого нет
		- Представитель: хара
