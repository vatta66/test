# Apicomplexa
подтип Sporozoa (cпоровики, apicomplexa) из типа [[Систематика Протист|Alveolata]]

[[Жизненные циклы Apicomplexa]]

- Для них характерны сложные жизненные циклы с чередованием гамогонии, спорогонии и мерогонии
- Все они - облигатные паразиты (т.е. нет другого способа жизни)
- Стадиии
	- Споры
		- Клетка, покрытая твердой оболочкой
		- Нужна для распространения
	- Зоита
		- Нужен для заражения клетки хозяина
		- Строение
			- Под мембраной 3 альвеолы
		- Моноэнергиды
		- Есть апикальный комплекс
			- Коноид
				- Служит для проникновения апикомплексы в клетку
				- Делает конец клетки жестким
			- Микронемы
				- Маленькие вакуоли
				- Синтезируют вещества и передают их в роптрии, встраиваясь в них
			- Роптрии
				- Большие вакуоли
				- Некоторые белки могут встраиваться в мембрану клетки и ослаблять [[Неспецифический иммунитет]] хозяина
				- Участвуют в создании паразитофорной вакуоли
					- В ней апикомплекса может жить и размножаться
					- Защищает от [[фагосома]] хозяина
		- Есть мионемы
			- Маленькие сократительные волоконца (актин+миозин)
			- Обеспечивают небольшое движение зоита
		- Есть микро- и макропоры
			- Микропорой питается, из макропоры выделяются вещества метаболизма
		- ![[Малярийный плазмодий-18-08-2024.png|300]]
- Классы
	- кл. Gregarinea
		- Грегарины
			- ![[Apicomplexa-01-10-2024.png|300]]
			- Соединение двух трофонтов грегарин - сизигий
	- кл. Coccidea
		- [[Токсоплазмы]]
	- кл. Haematozoa (кровеные споровики)
		- [[Малярийный плазмодий]]
