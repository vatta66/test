# Зародышевые оболочки

- Характерно для [[Пресмыкающиеся]], [[Птицы]], [[Млекопитающие]]
- Амнион - пузырь, в котором развивается эмбрион
	- Он заполнен амниотической жидкостью
	- Функции
		- Механическая защита
		- Защита от пересыхания
		- Препятствует проникновению инфекций к зародышу из полости матки
- Хорион (серозная оболочка)
	- Для связи с окружающей средой, участвует в жизнедеятельности зародыша
- Аллантоис
	- Для осуществления дыхания, накопления [[Продукты азотного обмена]] (мочевая кислота)
