# Алкадиены
Up: [[Органическая химия]]
Next:

Ациклические непредельные углеводороды CnH2n-2, 2 двойные связи

 Виды: 
 1. Кумулированные - двойная связь расположена подряд
 2. Сопряженные - 2-связи расположены через одну простую
	 1. Используются для промышенного получения каучуков (полимеров)
 3. Изолированные - между 2-связями более одной простой
	 1. Почти не отличаются по свойствам от [[алкены]]

 бутадиен-1,3(дивинил)
 2-метилбутадиен-1,3(изопрен)
 
### Методы получения алкадиенов
 
 - Метод Лебедева (этанол -> дивинил), кат: ZnO, Al2O3
 - Дегидрирование алканов (-2H2), кат: Cr2O3, Al2O3 → дивинил
 - Дегидратация диолов (гликолей)
	 - [[Алкадиены-24-03-2026.png]]
- Дегидрогалогенирование тетрагалогеналкана
	- Галогены через один атом углерода
- Способ Реппе
	- 2 формальдегида + ацетилен

### Хим св-ва алкадиенов

1. Галогенирование (+ Zn)
	1. При галогенировании 1 моль галогена: Раскрываются две двойные связи, по одному галогену присоединяется на каждый углерод(у которого были 2-связи), а так как у соседних атомов H освободилось по одной связи, они образуют двойную связь между собой
	2. При галогенировании 2 моль галогена: 2-связи раскрываются, и к каждому углероду, у которого освободилась связь присоединяются галогены.
2. Гидрирование(+H2), kat: Ni
	1. Получается алкен, тоже двойная связь по середине
3. Гидрогалогенирование
4. Гидратация → кетон
5. Перегруппировка
	1. Из алкадиена получаем алкин [[Алкадиены-31-03-2026.png]]
