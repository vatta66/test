# Системный подход в [[Научное познание]]

- Основная идея - целое не сводится к сумме частей, [[Эмерджентность]]
	- Нисходящая причинность
		- Свойства частей зависят от того, в какой системе они находятся
			- Данная идея опровергает [[Редукционизм]]
- Система - комплекс взаимодействующих элементов, объедененных определенной структурой
	- Для системы нужен системообразующий фактор, который будет вносить порядок
		- Например, для создания системы знаний это может быть [[Кластерообразующее знание]]
- [[Теория функциональных систем]] является последовательной в рамках данного подхода
