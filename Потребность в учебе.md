# Потребность в учебе

- У ученика должна быть [[Потребность]] в учении
- Ее можно создать с помощью
	- [[Проблемное обучение]]
	- [[Положительное подкрепление]]
- Школа должна создавать ученика, который будет скучать по учебе
- Когда ученик скучает по учебе, ощущает потребность в ней, он думает, что испытывает это без чьего-либо влияния (хоть это и не так)
	- Когда он начнет учиться, он будет чувствовать, что сделал это для утоления этого чувства, что будет являться реализацией [[Потребность в самодетерминации]], увеличивая [[Внутренняя мотивация]] ученика к учебе
