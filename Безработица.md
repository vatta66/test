# Безработица
[[Рынок труда]]

- **Безработица** - наличие в стране трудоспособных граждан, которые желают трудиться по найму при сложившемся уровне оплаты труда, но не могут трудоустроиться
- Критерии признания человека безработным
	- Трудоспособный возраст (с 16 до пенсионного)
	- Отсутствие постоянного заработка
	- Доказанное стремление найти работу (вы стоите на учете в центре занятости)
- Причины безработицы
	- [[Экономический цикл|Экономические кризисы]]
	- Чрезмерное использование политики [[Мировая экономика|фритредерства]]
	- Низкая [[Заработная плата]]
	- Демографический кризис
- Виды безработицы
	- Структурная
		- Спрос на профессии меняется (некоторые устаревают)
			- Поэтому важно [[life-long learning]]
		- Способы преодалеть
			- Программы переквалификации
			- Развитие [[Бизнес и предпринимательство|малого бизнеса]]
	- Циклическая
		- Обусловлена [[экономический цикл|экономическим спадом]] (кризисом, как великая депрессия)
		- Способы сокращения
			- Создание условий снижения предложения трудовых услуг (например, снижение пенсионного возраста)
			- Развитие малого [[Бизнес и предпринимательство|бизнеса]]/снижение [[Налоги]] для бизнеса
			- Создание условий роста спроса
			- Создание системы общественных работ
	- Фрикционная
		- Человек [[Расторжение трудового договора#^31d139|уволился по собственному желанию]] и ищет работу (т.е. на данный момент он безработный)
	- Естественная
		- Фрикционная+структурная, этот вид будет существовать всегда
		- Уровень безработицы при полной занятости
			- Полная занятость - уровень циклической безработицы = 0
	- Сезонная
		- Какая то профессия актуальная только в определенный сезон, когда он проходит - люди безработны (н. курорты)
		- Способ преодаления - программы общественных работ
- Последствия безработицы
	- Отрицательные
		- Рост [[Преступление|преступности]] и [[стресс]] (ухудшение физического состояния работников)
		- Недопроизводство [[Валовой внутренний продукт]]
		- Снижение дохода [[семья]]
	- Положительные
		- Трудовая [[мотивация]]
		- Снижение требований профсоюзов
			- Это одна из [[Причины инфляции]], так что это может помочь ее снизить
		- Формирование резерва рабочей силы
- Для уменьшения безработицы [[Государство]] проводит [[Политика государства по увеличению благосостояния страны]] и [[Экономическая политика государства]]

## План по теме “Безработица”
1. Понятие безработицы как социально-экономическое явление, связанное с рынком труда
2. Критерии признания человека безработным
	1. Трудоспособный возраст
	2. Доказанное желание работать (стоит на учете в центре занятости)
	3. Отсутствие постоянного заработка
3. Причины безработицы
	1. Экономические кризисы
	2. Чрезмерное использование политики фритредерства
	3. Низкая зарплата
	4. Демографический кризис
4. Виды безработицы
	1. Структурная
	2. Циклическая
	3. Фрикционная
	4. Сезонная
	5. Естественная
5. Положительные последствия безработицы
	1. Трудовая мотивация
	2. Снижения требований профсоюзов
	3. Формирование резерва рабочей силы
6. Отрицательные последствия безработицы
	1. Рост преступности и стресс
	2. Недопроизводство ВВП
	3. Снижение доходов семьи
