# Метанефридии

- Эволюционировали из [[протонефридии]]
- Ароморфоз [[Кольчатые черви]]

- Строение
	- Ресничная воронка
	- Выделительный канал + нефридиопор

- Состоит из воронки с ресничками (нефрострома), которая открывается во вторичную полость и собирает продукты обмена, и нефридиопоры - поры, которая открывается во внешнюю среду
- Стенки целома обвивают кровеносные сосуды
	- Этапы удаления продуктов метаболизма из крови
		- Ультрафильтрация
			- Фильтрование крови через базальную мембрану эпителия кровеносного сосуда (через [[коллаген]])
			- Через мембрану проходят аминокислоты, сахара, вода, а также продукты метаболизма
				- Все, что прошло через мембрану внуть целома, назыавется первичной мочей (ультрафильтрат)
		- Реабсорбция в метанефридии
- Собирает продукты распада из целома. Начинается в предыдушей части, выходит наружу в следующей. Обвит кров. сосудами для реабсорбции(возвращение полезных веществ обратно в [[Кровь]]) и секреции(удаление ненужных веществ в канал из крови).
