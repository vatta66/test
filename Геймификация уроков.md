# Геймификация уроков
идея по использованию [[Дидактические игры]]
- Сделать весь год в виде стратегической игры, где можно объединяться в команды, придумывать стратегии, добиваться личного успеха. Каждый урок можно превратить в битву между участниками игры (это поможет использовать [[Эффект Зейгарник]]). По окончании битвы можно выдавать ресурсы, из которых команды смогут что-то делать, допустим, выстраивать защиту, создавать армию для наступления на другие команды, прокачивать солдат и стены и тд. 
- Таким образом я предлагаю геймефицировать весь учебный год
- Вопрос в том, насколько правильно использовать такую тотальную [[Внешняя мотивация]]
- По сути, в этом заключается главная проблема любой геймификации.
	- Но я думаю, что если ввести такую практику, а потом аккуратно подменить воображаемые игры на игры внутри науки, то есть на поиск пазлов для единой картины, то проблема решится
		- Если рассматривать науку в целом с точки зрения сети(графа), то у нас получается геймификация науки
		- Например, мы можем обозначить для учеников проблему(допустим, влияние сахара на продолжительность жизни/умственные способности и тд), и для решения этой проблемы они вместо того, чтобы руководствоваться мотивацией улучшить свою армию, узнают новое воимя решения проблемы(которая для них актуальна). Главное правильно ее обозначить
		- Добавление новых элементов в граф знаний само по себе может являться хорошей мотивацией
