# Млекопитающие
Класс Звери из подтипа [[Позвоночные]]
![[Млекопитающие-22-06-2025.png|724]]
[[Млекопитающие.excalidraw]]

- [[Ароморфозы млекопитающих]]
- Строение
	- Кости черепа соединены швами
	- Коракоид (воронья кость) отсутствует
		- Кроме однопроходных (ехидна, утконос)
	- В коже выражена жировая клетчатка - термоизоляция
	- Есть производные эпидермиса - когти, ногти, железы и волосы
	- Сквозная пищеварительная система
		- Нет клоаки, вместо нее - анальное отверстие
			- Клоака есть у однопроходных
		- У [[растительноядные млекопитающие]] есть спец. приспособления
	- Выделительная система - тазовые [[почки]]
		- Это [[Мочевыделительная система человека]]
	- Нервная система трубчатого типа
		- Максимальное развитие
			- Забота о потомстве, общение особей и [[Условный рефлекс]]
		- [[Строение уха|Уши]] развиты очень хорошо
			- Появляется [[Строение уха|внешнее ухо]], ушная раковина, три слуховые косточки в среднем ухе (стремечко, наковальня, молоточек)
	- Есть [[Зародышевые оболочки]]
- Произошли от цинодонтов
