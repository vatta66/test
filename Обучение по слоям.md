# Обучение по слоям

- Вся информация распределяется по трем уровням
	- Слой 1 - Структура. Базовые концепты
	- Слой 2 - Информация, помогающая понять базовые концепты, немного расширяющая ее
	- Слой 3 - информация, дополняющая и уточняющая слой 2
- Пошагово
	- Выпишите все термины
	- Все термины темы распределяются по 3 уровням
	- Составьте дерево(сост. из слоев) из терминов в соответствии с их уровнями
- Нельзя приступать к освоению 3 слоя какого-то из терминов, не освоив весь второй слой темы
- Отношение к [[Концентрическая структура изложения(педагогика)]]
	- Концентрическая структура старается соблюдать этот принцип, формируя у ученика базу, а затем, через какое то количество времени, добавляя новые слои
	- Но делает она это неэффективно, т.к. базу приходится вновь получать из-за слишком большого промежутка между освоением слоя 1 и 2 и освоением слоя 3
	- [[Спиральная структура изложения материала]] для реализации этой практики подходит лучше
