# Ситовидные элементы
Up: [[Флоэма]]
Next:

- Похожи на [[ксилема|трахеальные элементы]], главное отличие - у них сохраняется мембрана
	- Также, у них есть немного [[Митохондрия]], измененных [[Пластиды]], [[Эндоплазматическая сеть]]
- На месте групп плазмодесм возникают сквозные ситовидные отверстия
	- Это происходит из-за эрозии клеточной стенки, когда в нее откладывается полисахарид каллоза
- Виды
	- Ситовидные клетки
		- У споровых и [[Голосеменные]]
		- Длинные клетки без вакуоли, с неактивным (пикнотическим) ядром
			- Поэтому функции ядра берут на себя клетки, окружающие ситовидную - клетки страсбургера
			- Они соединены с ситовидной плазмодессами
	- Ситовидные трубки с клетками-спутницами
		- Клетки ситовидной трубки связаны через ситовидные пластинки (часть с большим количеством ситовидных отверстий)
			- Ситовидные отверстия окаймлены каллозой
			- ![[Флоэма-15-08-2025 1.png|735]]
		- Клетки-спутницы
			- Есть лейкопласты без крахмала
			- Митохондрии образуют сеть 
			- Функции: обеспечить энергией ситовидные трубки, загрузить в них все необходимое
			- Виды в зависимости от контактов с другими клетками
				- Открытые спутники
					- Пронизана тысячами плазмодесм с двух сторон - таким образом, что клетка ситовидной трубки оказывается соединена с другими клетками через систему ЭПС клетки-спутницы
						- ЭПС протянута от паренхимных клеток, через плазмодесмы спутников в ситовидные трубки
						- ![[Флоэма-15-08-2025 3.png|229]]
					- На каждую клетку ситовидной трубки один открытый спутник
				- Закрытые спутники
					- Плазмодесмы есть только между спутником и клеткой ситовидной трубки
					- А контакт с другими клетками происходит через межклеточную среду
						- Для этого у такого спутника клеточная стенка принимает извилистую форму - это увеличивает ее поверхность для транспорта
						- ![[Флоэма-15-08-2025 2.png|259]]
					- На одну клетку ситовидной трубки по несколько закрытых спутников
