# Формирование долговременной памяти

Для формирования [[Ассоциативное обучение]] нужно чтобы сенсорный сигнал постоянно присутствовал, т.е. это несколько часов.
Чтобы **быстро сформировать ассоциотивную память**, нужно к схеме обучения добавить запись в круг Пейпеза: ![[Pasted image 20220110200007.png]]
Во время сенсорного стимула будут выбиваться магниевые пробки на входе в [[Гиппокамп]], откуда информация начнет циркулировать по кругу Пейпеца. Дальше, когда стимул исчезнет, его будет заменять сигнал из круга Пейпеза, активируя механизм ассоциотивного обучения. 
Активно перезапись информации из Круга пейпеза(кратковременная память) в долговременную(влияние на обучающиеся нейроны) идет во время REM-сна
