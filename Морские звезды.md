# Морские звезды
Up: [[Иглокожие]]
Next:

- Строение
	- Центральный диск, от которого по радиальным лучам отходят руки 
	- Экзоскелет
		- Педицеллярии – известковые выросты в виде щипцов
			- Для того, чтобы снимать с поверхности тела мусор, [[Паразитизм|паразитов]] и их личинок
			- ![[Морские звезды-22-05-2025 1.png|300]]![[Морские звезды-22-05-2025 2.png|300]]
		- Паксиллы
			- Выросты в виде зонтиков
			- Нужны для защиты папул, которые находятся между ними
	- Амбулакральные склериты с ампулами
- Анатомия
	- Пищеварительная система
		- Ротовое отверстие
		- Кардиальный желудок
			- Из 5 карманов, которые выворачиваются через ротовое отверстие наружу, когда звезда хочет кого-то захамячить
		- Пилорический желудок
			- Тонкий, с парными пилорическими выростами на каждую руку
				- Вырабатывают ферменты
		- Задняя кишка
			- Короткая, ее окружают ректальные железы (не понятно, зачем нужны)
		- Задняя кишка с анальным отверстием
	- ![[Морские звезды-22-05-2025.png|300]]
- Развитие
	- Диплеврула → бипиннария (плохо развиты руки) → Брахиолярия (хорошо развиты руки) → взрослая особь
- Питание морских звезд
	- ![[Морские звезды-25-05-2025.png|654]]
	- Охотятся на [[Двустворчатые моллюски]]
	- Пытются раскрыть их створки, могут закрепить свои конечности в определенном положении благодаря регулировке жесткости коллагена
		- Моллюск же всегда прикладывает усилия, чтобы держать створку закрытой
		- Так что звезда ждет и съедает его
