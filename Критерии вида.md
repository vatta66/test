# Критерии вида
Up:
Next:

- Морфологический
	- Самый древний
	- Сравнение внешних сходств и различий
	- Но он ненадежен, так как не может различать видов-двойников
		- Они морфологически сходны
	- ![[Pasted image 20231008234156.png]]236
- Кариотипический
	- Сходства и различия хромосомного набора
- Молекулярно-генетический
	- Создается международная библиотека штрихкодов [[ДНК]]
	- Для определения видов животных используется анализ [[митохондрия|митохондриальной]] [[ДНК]]
		- Для растений - ДНК [[Хлоропласты]]
			- Т.к. один хлоропласт может содержать до 100 копий хлоропластного генома - его проще изучать
		- Для [[бактерии]] и [[простейшие]] - [[ген]]ы, кодирующие синтез рибосомальной РНК
- Экологический
	- Какую экологическую нишу занимает особь
	- Чем питаются, где живут (не конкретное название, а местность), условия климата, куда отклаыдвают яйца
		- + для растений, светолюбивое оно или нет 
	- Показывает взаимодействие особи с [[Экологический фактор|факторами среды - биотическими, абиотическими]]
	- ![[Pasted image 20231008234055.png|638]] 125
	- ![[Pasted image 20231008234309.png]] 456
- Географический
	- Анализируется ареал обитания особей
	- В нем есть конкретное географическое название
	- Ненадежный критерий, т.к. ареалы могут перекрываться
	- ![[Pasted image 20231008233957.png]] 136
- Биохимический
	- Сходство состава белков, нуклеиновых кислот и [[Метаболизм]]а
	- Часто используется для определение видов бактерий
		- Например, их выращивают на различной питательной среде и смотрят, как усваиваются эти питательные элементы, какие продукты отхода
	- ![[Pasted image 20231008234431.png]] 456
- Физиологический
	- Особенности процессов жизнедеятельности организма и отдельных систем органов
	- Все что связано с [[Виды размножения|размножение]]
	- ![[Pasted image 20231008233814.png]]345
	- ![[Pasted image 20231008234352.png]] 456
