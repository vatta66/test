# Отбор [[ген]]ов идет по

- Умению генов предсказывать будущее
	- В дальнейшем - созданию мозга, эффективно способного моделировать будущее - [[Мозг пытается предсказывать будущее]]
- Способности машины для выживание использовать свою среду обитания
	- В конечном итоге инструкция генов мозгу становится "делай, что хочешь, для нашего выживания"
- Если наличие кроссинговера определяется геном и если кроссинговер выгоден данному гену, то этого достаточно для объяснения существование кроссинговера
		- Такой [[ген]] распространится в генофонде
	- То же самое и с половым и бесполым размножением
	- Благоприятно ли это изменение для остальных генов - неважно ([[Теория эгоистичного гена]])
