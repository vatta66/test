# Серотонин

- Создание: Триптофан -> 5-гидрокситриптофан(5-HTP) -> серотонин(5-гидрокситриптамин)
	- Синтез идет с помощью [[Триптофангидроксилаза]]

Существует 7 типов рецепторов 5-HT (5-HT1 - 5-HT7)
- 5-HT1 - тормозят [[Аденилатциклазная система]] - торможение мозга
- 5-HT2 - активирует фосфолипазу C (фермент, создающий ВтП для закрывания [[K-электрочувствительный канал]] - вызывает возбуждение постсинапт. клетки)

- Серотонин как [[Гормоны]] в большей степени выделяют [[тромбоциты]]
	- Для сужения сосудов (уменьш. кровотеч.)
- Есть гипотеза связи [[депрессия]] с серотонином, но пока нормально не доказана
- [[Серотонин в ядрах шва]]
- Серотонин работает в том числе как внутренний тормоз, ослабляя сиюминутные порывы
	- Similar to [[Дорсолатеральная префронтальная кора]]
	- [[Недостаток серотонина]]
