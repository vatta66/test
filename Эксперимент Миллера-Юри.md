# Эксперимент Миллера-Юри
Up: [[Происхождение жизни]] - [[Эволюция]]

- Они сделали стерильную установку, внутри которой находились вещества, повторяющие состав древней атмосферы Земли: метан, аммиак, водород, монооксид углерода, отсутствие кислорода
- Далее они создали определенные условия: температура, электрические заряды, водная среда (то есть получился своеобразный симулятор атмосферы древней Земли
- В результате были получены простейшие органические кислоты и аминокислот
