# Центральность по посредничеству

- Отражает то, насколько сильно один узел сети связан с другими узлами, не связанными друг с другом напрямую
	- В таком случае, если из группы убрать данный узел, то она распадется
	- В таком случае он может 
		- Организовывать взаимодействие этих узлов, не опасаясь за то, что они будут чего-то недоговаривать
		- Влиять нужным для себя образом на их взаимодействие
- Если на вечеринке большинство гостей знает только вас, то скорее всего вы станете ее центром
	- Если же на вечеринке все знают друг друга, то [[общение]] будет менее централизованным
