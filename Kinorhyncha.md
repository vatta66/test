# Kinorhyncha
Up: [[Схизоцельные черви]]
Next:

- Тело из 12 сегментов
	- На некоторых есть длинные щетинки - для фиксации тела в субстрате
- Живут между песчинками морского грунта
	- Такие организмы называются интерстициальные
		- Например, еще [[трихоплакс]]
		- Может питаться [[Фораминиферы]]
			- Еще и для этого нужны скалиды - чтобы дробить их оболочку
- Пищеварительная система
	- Очень мускулистая глотка
- Протонефридиальная система
	- Циртоциты начинаются в предыдущем отделе, выводятся в следующем
- Нервная система - брюшная нервная цепочка
	- Сближенные до предела коннективы
