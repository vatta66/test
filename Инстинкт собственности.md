# [[Инстинкт]] собственности

- Один из предполагаемых [[Инстинкты человека]]
- Выражается в защите собственности ([[Вещь в праве|имущество]])
	- [[Агрессия]]
- У каких-то людей этот инстинкт выражен сильнее
	- Возможно, что это связано с излишней активностью [[Миндалина]]
- Эксперементальная проверка
	- Измерение активность миндалины разных людей в смоделированных условиях
	- Найти [[ген]]ы, коррелирующие с повышенной активностью миндалины в данных условиях, тем самым подтверждая наличие [[Инстинкт]]а
