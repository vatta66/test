# Классификационное знание

- Знание, которое является [[Классификация]] определенного количества [[Понятие]]
- Желательно, чтобы классификационное знания формировалось учениками самостоятельно с помощью сравнения и сопоставления уже полученных [[Декларативное знание]], входящих в [[Объем понятия в теории системного обучения|объем каждого понятия]] из классификации
	- Например, с помощью 
		- [[Эвристическая беседа]]
		- [[Лабораторная работа по принципу проблемного обучения]]
- Включает в себя умения
	- [[Анализ]]
	- [[Выделение главного]]
	- [[Сравнение]]
	- [[Создание классификации]]
- В основном, в [[Кластер знаний]] классификации знания будут связаны между собой [[Компаративная связь]]
- Принципы:
	- [[Кластеры знаний, находящиеся в концептуальном знании, должны быть достаточно сформированы]]
	- [[У ученика должен быть кластер знаний о принципе классификации для каждого знания, входящего в классификацию]]
