# Уровень негатива в различных ситуациях

- U-индекс [1]
	- A part of time when person feel [[Негативные эмоции]]
- Американки испытывают неприятные ощущения примерно 19 % времени — немногим больше, чем француженки (16 %) или датчанки (14 %) ([pdf](zotero://open-pdf/library/items/LU9XSCWF?page=401&annotation=Q7URBV6Q))
- Для 1000 американок в городе на Среднем Западе U-индекс составил 29 % для дороги на работу, 27 % — для работы, 24 % — для забот о детях, 18 % — для домашних дел, 12 % — для общения, 12 % — для просмотра телепередач и 5 % — для [[Секс]]а ([pdf](zotero://open-pdf/library/items/LU9XSCWF?page=402&annotation=NS3END6S))
	- Американки от общения с детьми получают чуть меньше удовольствия, чем от домашней работы. Тут обнаружилось одно из немногих расхождений между американками и француженками: француженки меньше времени проводят с детьми, но получают больше удовольствия от общения с ними; возможно, потому, что к их услугам больше детских учреждений и меньше приходится по вечерам возить детей на всякие занятия ([pdf](zotero://open-pdf/library/items/LU9XSCWF?page=402&annotation=32NMHWKN)) - [[Воспитание]]
