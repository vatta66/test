# Типы личинок [[насекомые|насекомых]]
Up: [[Развитие насекомых]]
Next:

Какие есть классификации личинок?

- По конечностям
	- Олигоподные (с 3 парами грудных конечностей)
	- Протоподные (только зачатки конечностей, как бугорки)
	- Полиподные (3 пары грудных + 2-8 ложные брюшные)
		- Ложные, потому что не членистого строения - просто выросты тела
		- Например, гусеницы бабочек
	- Аподные (нет грудных ног)
		- Двукрылые, некоторые жуки
- По внешнему виду
	- Камподеовидные
		- Удлиненный последний сегмент абдомена - для отталкивания
		- Сильно склеретизованное тело, есть церки
		- ![[Развитие насекомых-15-05-2025.png|295]]
	- Эруковидные
		- С-образная форма тела, тело слабо склеретизовано
		- Личинки майских жуков, муравьев
		- ![[Развитие насекомых-15-05-2025 1.png|248]]
	- Проволочниковые
		- Очень сильно хитинизированные
		- Короткие мочные конечности
		- Круглое в поперечном сечении тело
		- Часто у жуков
			- ![[Развитие насекомых-15-05-2025 2.png|300]] - личинка жука-щелкуна
	- Червеобразные
		- Аподные или с очень короткими конечностями
		- Слабо склеретизированное тело, слабо выражена голова
		- У жуков, двукрылых
		- ![[Развитие насекомых-15-05-2025 3.png|239]]
	- Гусенициы
