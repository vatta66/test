# Поясковые черви
Надкласс Citeliata

- Гермафродиты
	- Способ размножения - синхронный гермафродитизм
		- Два червя встречаются и синхронно оплодотворяют друг друга
			- В отличие от [[Плоские черви]], где только один оплодотворяет другого
- Копуляция
	- Два червя слипаются друг с другом с помощью секретов пояска
		- Кусок кожи, содержащий множество слизевых желез
	- Покрывают себя коконом из слизи
	- Далее метают в него свои сперматозоиды так, чтобы другой червь мог уловить их своими семяприемниками
	- Они разрываются, унося с собой по половине кокона
	- Внутрь кокона выделяются яйцеклетки
	- Кокон начинает сползать в сторону головы, где находятся семяприемники
		- Сперматозоиды вбрасывается внутрь кокона, где они оплодотворяет яйцеклетки
	- Червь скидывает кокон, где образуются яйца
		- Для них кокон - защитная оболочка
- Lumbricus terrestris (дождевой червь)
	- Поедают органику
		- Переваренная ими органика усваивается растениями лучше, чем обычная - выполняют функцию [[Трофические уровни|редуцентов]]
		- Важные почвообразователи
