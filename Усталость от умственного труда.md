# Усталость от умственного труда

- Причина - Избыток глутамата при усиленной умственной деятельности
- При длительной усиленной умственной деятельности в [[Синапс|синаптических щелях]] [[Префронтальная кора]] начинает накапливаться [[Глутамат]]
	- [[Глиальные клетки|Астроциты]] не успевают выкачивать его оттуда
	- Глутамата для обмена сигналами начинает нехватать
	- Глутомату, который производят нейроны, трудно пробиться через забитые синаптические щели
- Мозг начинает хуже работать
- [[Усталость повышает креативность]]
