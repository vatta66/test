# Синапс

- Контакт аксона одного [[Нейрон]] с дендритом другого
- Елементарная ед функционирования мозга
- По нейронам информация подается в электрической форме, а между нейронами - в химической(с помощью [[Нейромедиатор]])
- В окончании аксона может происходить [[Синтез нейромедиатора]]
- Присутсвует **насос обратного захвата**, который осуществляет [[Инактивация нейромедиатора]]
