# Трансформация
Up: [[Горизонтальный перенос генов]]

- [[Бактерии]] могут поглощать [[ДНК]] из окружающей среды и встраивать ее себе
	- Получая тем самым различные новые признаки
- Для того, чтобы осуществить трансформацию, бактерия должна быть компетентна
	- То есть готова к тому, чтобы принять ДНК через свою [[клеточная мембрана]]
	- Компетентность клетки может активировать [[чувство кворума]]
