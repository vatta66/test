- Диморфная колония
- Медузы, образующиеся в гонангиях, выходят в свободное плавание
	- Число щупалец - 4 или кратно 4
	- Наружная часть зонтика -  эксумбрелла, внутренняя - субумбрелла
	- Рот в виде хоботка
	- После него - желудок и радиальные каналы
		- Кол-во радиальных каналов = числу жгутиков
		- Впадают в кольцевой канал покругу медузы
	- Есть складка - парус
		- Когда расширяется - запускает воду из субумбреллы, когда сужается - выбрасывает
		- Обеспечивает реактивное движение
	- Также внутри есть гонады
- Жизненный цикл
	- Полип отпочковывает медуз
	- Медузы создают гаметы в гонадах, которые копулируют в толще воды
	- Из зиготы развивается целобластула
	- Из нее - паренхимула
		- В результате иммиграции клеток снаружи целобластулы внутрь
	- Из нее - планула
		- Т.к. клетки внутри паренхимулы выстилаются по краям, образуя энтодерму
	- Из планулы развивается одиночный полип по типу гастрозоида
	- Из него с помощью незавершенного почкования образуется колония
