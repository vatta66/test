# Размножение [[грибы|грибов]]
Up:  [[Виды размножения]]
Next:

- Вегетативное
	- Участками [[Мицелий]]я
	- Почкованием
	- Артроспорами (оидиями)
		- Клетки с тонкой оболочкой, на которые распадается мицелий
		- Например, у [[Голосумчатые грибы]]
	- хламидоспорами
		- Клетки с толстой оболочкой, могут долго переносить неблагоприятные условия
- Бесполое
	- С помощью спор, которые образуются на специальных органах
		- Это главное отличие от вегетативного способа
	- Эндогенные споры
		- Образуются внутри спец **клеток**
			- Зооспорангии - если в ней образуются подвижные споры
			- Спорангий - если неподвижные
		- характерны для [[Низшие грибы]]
	- Экзогенные споры - конидии
		- Образуются на конидиеносце - гифа, поднимающаяся вертикально
		- ![[Размножение грибов-26-03-2025.png|300]]
			- То есть большая штука снизу - конидиеноцес, а маленькие наверху - конидии (это сами споры)
