# Мой проект по внедрению системы [[цеттелькастен]] в школы

- Внедрение цеттелькастена в систему образования поможет развить у школьников [[Метакогнитивное знание|метакогнитивные навыки]] (актуальность данного развития?) и улучшить когнитивные
- Цеттелькастен в школах как средство реализации [[Проблемно-комплексная теория]]
- Также данный метод подойдет для реализации [[Проблемное обучение]]. Выглядеть это будет следующим образом:![[Pasted image 20221010221308.png]]6 - проблема; 1,2 - факты; 3,4 - идея ученика, возникшая на основе этих фактов; 5 - [[Синтез (мысл.о.)]] идей ученика в новое знание. Пункт 5 может быть пропущен, т.к. результатом синтеза может стать непосредственно проблема. Количество проблем и входных теорий может быть разное
- [[Способ системного ведения конспекта в тетради]]
- Развитие [[Навык системного мышления]]
- Из чего это следует:
	- [[Обучение через иерархию межпредметных связей]]
- Реализация [[Принцип систематичности и последовательности]] в школах через цеттелькастем
	- Также возможно применение [[Системная спираль изложения]]
