# Исследовательский метод (педагогика)

- Описание
	- Учитель формулирует проблему и ситуативно контролирует процесс
	- Решению проблемы отводится целый блок времени ([[Решение проблемы]])
	- [[Атомарное знание]] вообще не даются ученикам, они сами добывают их в поиске решения
	- Процесс сопровождается повышенной интенсивностью и [[Познавательный интерес]] учащихся
		- Следовательно, данный метод является [[Средства для формирования познавательного интереса]]
- Недостатки
	- Большие временные затраты времени и энергии учителя и учащихся
